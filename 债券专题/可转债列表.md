# 可转债列表（MCP 工具 `ft_get_cb_lists_handler`）

> **MCP 工具**：`ft_get_cb_lists_handler`（category: `债券专题`）。返回 **markdown 表格文本**（非结构化 JSON）。输入参数 / 输出参数 / 数据样例对照 [ftshare-doc](../../ftshare-doc/api-doc) 原接口。

- 描述：获取可转债列表（债券全称、债券代码、正股代码、交易所）。无入参，返回 `CbListsResponse`。数据为前一交易日可转债基础数据(v3) `V3_1`，由 `get_cb_lists` 读取。
- 数据范围：最新快照（前一交易日可转债基础数据，无时间序列）
- 单次限量：全量列表返回，无分页
- 提示：
  - 无入参，请求体为空对象。
  - 返回全市场可转债清单。

## 输入参数

无入参（请求体为空对象 `CbListsRequest {}`）。

## 输出参数

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| items | array[CbListsItem] | Y | 可转债列表 |

CbListsItem：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| full_name | string | Y | 债券全称 |
| cb_id | int | Y | 债券代码 |
| stock_id | int | Y | 转债对应正股代码 |
| exchange | int | Y | 交易所 |

## 调用方法（MCP）

> MCP 工具名 `ft_get_cb_lists_handler`。MCP Streamable HTTP 要求**先 initialize 拿 `Mcp-Session-Id`，再 `tools/call`**，后续请求带该 header。返回 **markdown 表格文本**。

**curl**：

```bash
SID=$(curl -sS -m 10 -D - -o /dev/null -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"t","version":"1"}}}' \
  | grep -i 'mcp-session-id:' | awk '{print $2}' | tr -d '')

curl -sS -m 10 -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -H "Mcp-Session-Id: $SID" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ft_get_cb_lists_handler","arguments":{}}}'
```

**Python（`mcp` SDK，自动握手管理 session）**：

```python
import asyncio
from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client

async def main():
    async with streamablehttp_client('<MCP_BASE_URL>') as (r, w, _):
        async with ClientSession(r, w) as s:
            await s.initialize()
            res = await s.call_tool('ft_get_cb_lists_handler', {})
            print(res.content[0].text)   # markdown 表格文本

asyncio.run(main())
```
