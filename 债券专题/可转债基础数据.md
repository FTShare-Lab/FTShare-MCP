# 可转债基础数据（MCP 工具 `ft_get_cb_base_data_handler`）

> **MCP 工具**：`ft_get_cb_base_data_handler`（category: `债券专题`）。返回 **markdown 表格文本**（非结构化 JSON）。输入参数 / 输出参数 / 数据样例对照 [ftshare-doc](../../ftshare-doc/api-doc) 原接口。

- 描述：按转债代码查询单只可转债基础信息（票面/付息/转股/赎回/上市等要素）。handler 校验 `symbol_code` 非空，命中返回单条 `CbBaseDataResponse`，未找到返回错误。数据为前一交易日可转债基础数据(v3) `V3_1`，由 `get_cb_base_data` 读取。
- 数据范围：最新快照（前一交易日可转债基础数据，无时间序列）
- 单次限量：单只转债，单条返回
- 提示：
  - `symbol_code` 必填。
  - 枚举字段以字符串返回：`bond_type`（cb/eb/STb）、`trade_type`（dirty_price/clean_price）、`coupon_method`（stepup_rate/fixed_rate）。
  - 上市/转股相关字段未上市或未发生转股时为 `null`。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| symbol_code | string | Y | 转债代码 |

## 调用方法（MCP）

> MCP 工具名 `ft_get_cb_base_data_handler`。MCP Streamable HTTP 要求**先 initialize 拿 `Mcp-Session-Id`，再 `tools/call`**，后续请求带该 header。返回 **markdown 表格文本**。

**curl**：

```bash
SID=$(curl -sS -m 10 -D - -o /dev/null -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"t","version":"1"}}}' \
  | grep -i 'mcp-session-id:' | awk '{print $2}' | tr -d '')

curl -sS -m 10 -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -H "Mcp-Session-Id: $SID" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ft_get_cb_base_data_handler","arguments":{"symbol_code": "113001"}}}'
```

**Python（`mcp` SDK，自动握手管理 session）**：

```python
import asyncio
from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client

async def main():
    async with streamablehttp_client('<MCP_BASE_URL>') as (r, w, _):
        async with ClientSession(r, w) as s:
            await s.initialize()
            res = await s.call_tool('ft_get_cb_base_data_handler', {"symbol_code": "113001"})
            print(res.content[0].text)   # markdown 表格文本

asyncio.run(main())
```
