# 基金净值（MCP 工具 `ft_get_fund_nav`）

> **MCP 工具**：`ft_get_fund_nav`（category: `公募基金`）。返回 **markdown 表格文本**（非结构化 JSON）。输入参数 / 输出参数 / 数据样例对照 [ftshare-doc](../../ftshare-doc/api-doc) 原接口。

- 描述：按基金代码分页查询基金净值序列（单位净值/累计净值/复权净值/日增长率/每万份收益/7日年化/申赎状态/分红配送）。返回 `PaginatedResponse<FundNavResponse>`，handler 校验 `institution_code` 非空，默认 `page=1`、`page_size=50`。数据经 Redis 缓存 + ClickHouse 回源。
- 数据范围：以服务端返回为准
- 单次限量：分页返回，默认 `page=1`、`page_size=50`
- 提示：
  - `institution_code` 必填。
  - 时间序列数据，按 `end_date`（截止日期）排序分页。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| institution_code | string | Y | 基金代码 |
| page | int | N | 当前页，默认 1 |
| page_size | int | N | 每页条数，默认 50 |

## 输出参数

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| items | array[FundNavResponse] | Y | 净值列表 |
| total_pages | int | Y | 总页数 |
| total_items | int | Y | 总记录数（未分页前） |

FundNavResponse：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| security_code | string | Y | 基金代码 |
| security_name | string | Y | 基金名称 |
| end_date | int | Y | 截止日期 YYYYMMDD |
| nv | decimal | Y | 资产净值 |
| unitnv | decimal | Y | 单位净值 |
| unitnv_grw | decimal | Y | 单位净值日增长率（%） |
| accum_unitnv | decimal | Y | 累计净值 |
| prof_d | decimal | Y | 每万份基金收益（元） |
| annurtn_7d | decimal | Y | 7日年化收益率 |
| subscribe_status | string | Y | 申购状态 |
| redeem_status | string | Y | 赎回状态 |
| dividend | string | Y | 分红配送 |

## 调用方法（MCP）

> MCP 工具名 `ft_get_fund_nav`。MCP Streamable HTTP 要求**先 initialize 拿 `Mcp-Session-Id`，再 `tools/call`**，后续请求带该 header。返回 **markdown 表格文本**。

**curl**：

```bash
SID=$(curl -sS -m 10 -D - -o /dev/null -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"t","version":"1"}}}' \
  | grep -i 'mcp-session-id:' | awk '{print $2}' | tr -d '')

curl -sS -m 10 -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -H "Mcp-Session-Id: $SID" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ft_get_fund_nav","arguments":{"institution_code": "110011"}}}'
```

**Python（`mcp` SDK，自动握手管理 session）**：

```python
import asyncio
from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client

async def main():
    async with streamablehttp_client('<MCP_BASE_URL>') as (r, w, _):
        async with ClientSession(r, w) as s:
            await s.initialize()
            res = await s.call_tool('ft_get_fund_nav', {"institution_code": "110011"})
            print(res.content[0].text)   # markdown 表格文本

asyncio.run(main())
```
