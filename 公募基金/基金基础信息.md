# 基金基础信息（MCP 工具 `ft_get_fund_basicinfo`）

> **MCP 工具**：`ft_get_fund_basicinfo`（category: `公募基金`）。返回 **markdown 表格文本**（非结构化 JSON）。输入参数 / 输出参数 / 数据样例对照 [ftshare-doc](../../ftshare-doc/api-doc) 原接口。

- 描述：按基金代码查询基金基本信息（管理人/托管人/运作方式/投资目标/费率要素等静态档案）。handler 校验 `institution_code` 非空，命中返回单条 `FundBasicinfoResponse`，未命中返回错误。数据由 `query_fund_basicinfo` 经 Redis 缓存 + ClickHouse 回源。
- 数据范围：最新快照（静态档案，无时间序列）
- 单次限量：单只基金，单条返回
- 提示：
  - `institution_code` 必填（基金代码）。
  - `page` / `page_size` 在本接口未参与基础信息查询，仅与同 Request 类型复用相关。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| institution_code | string | Y | 基金代码 |
| page | int | N | 当前页（本接口未实际使用分页） |
| page_size | int | N | 每页条数（本接口未实际使用分页） |

## 调用方法（MCP）

> MCP 工具名 `ft_get_fund_basicinfo`。MCP Streamable HTTP 要求**先 initialize 拿 `Mcp-Session-Id`，再 `tools/call`**，后续请求带该 header。返回 **markdown 表格文本**。

**curl**：

```bash
SID=$(curl -sS -m 10 -D - -o /dev/null -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"t","version":"1"}}}' \
  | grep -i 'mcp-session-id:' | awk '{print $2}' | tr -d '')

curl -sS -m 10 -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -H "Mcp-Session-Id: $SID" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ft_get_fund_basicinfo","arguments":{"institution_code": "110011"}}}'
```

**Python（`mcp` SDK，自动握手管理 session）**：

```python
import asyncio
from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client

async def main():
    async with streamablehttp_client('<MCP_BASE_URL>') as (r, w, _):
        async with ClientSession(r, w) as s:
            await s.initialize()
            res = await s.call_tool('ft_get_fund_basicinfo', {"institution_code": "110011"})
            print(res.content[0].text)   # markdown 表格文本

asyncio.run(main())
```
