# 公告列表（MCP 工具 `ft_stock_announcements`）

> **MCP 工具**：`ft_stock_announcements`（category: `大模型语料`）。返回 **markdown 表格文本**（非结构化 JSON）。输入参数 / 输出参数 / 数据样例对照 [ftshare-doc](../../ftshare-doc/api-doc) 原接口。

- 描述：查询 A 股公告列表，底层从 ClickHouse `ashare_announcements` 表查询并通过 Redis 缓存。同一路径两种模式：① **按标的**——`stock_code` 存在时查询单票所有公告；② **按日期范围**——`start_date` 存在时查询指定日期范围内全市场公告，日期范围校验不能超过 3 天。返回 `PaginatedResponse`（items/total_pages/total_items）。
- 数据范围：以服务端返回为准
- 单次限量：分页返回，`page` / `page_size` 必填；按日期范围查询时区间跨度不能超过 3 天
- 提示：
  - 二选一：必须提供 `stock_code`（按标的查）或 `start_date`（按日期范围查），否则报错。
  - `type` 当前只支持 `"stock"`。
  - 按日期范围查询时，`start_date` / `end_date` 格式 `YYYYMMDD`，区间不能超过 3 天；`end_date` 不填默认当前时间。
  - 公告正文文件下载请用 `/api/v1/market/data/announcements/stock-announcements/{url_hash}`（另见独立接口）。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| stock_code | string | N | 证券代码（按标的查询时必填） |
| start_date | string | N | 开始日期 YYYYMMDD（按日期范围查询时必填） |
| end_date | string | N | 结束日期 YYYYMMDD，不填默认当前时间 |
| type | string | Y | 查询类型，当前只支持 `stock` |
| page | int | Y | 页码 |
| page_size | int | Y | 每页条数 |

## 输出参数

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| items | array[AnnouncementItem] | Y | 公告列表 |
| total_pages | int | Y | 总页数 |
| total_items | int | Y | 总记录数（未分页前） |

AnnouncementItem：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| stock_code | string | Y | 证券代码 |
| stock_name | string | Y | 证券名称 |
| announcement_id | string | Y | 公告 id |
| announcement_title | string | Y | 公告标题 |
| announcement_time | string | Y | 公告时间，格式 YYYY-MM-DD HH:MM:SS |
| column_type | string | Y | 栏目类型 |
| url_hash | string | Y | 公告文件 URL 哈希，用于下载正文文件 |

## 调用方法（MCP）

> MCP 工具名 `ft_stock_announcements`。MCP Streamable HTTP 要求**先 initialize 拿 `Mcp-Session-Id`，再 `tools/call`**，后续请求带该 header。返回 **markdown 表格文本**。

**curl**：

```bash
SID=$(curl -sS -m 10 -D - -o /dev/null -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"t","version":"1"}}}' \
  | grep -i 'mcp-session-id:' | awk '{print $2}' | tr -d '')

curl -sS -m 10 -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -H "Mcp-Session-Id: $SID" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ft_stock_announcements","arguments":{"type": "stock", "stock_code": "600519.SH", "page": 1, "page_size": 3}}}'
```

**Python（`mcp` SDK，自动握手管理 session）**：

```python
import asyncio
from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client

async def main():
    async with streamablehttp_client('<MCP_BASE_URL>') as (r, w, _):
        async with ClientSession(r, w) as s:
            await s.initialize()
            res = await s.call_tool('ft_stock_announcements', {"type": "stock", "stock_code": "600519.SH", "page": 1, "page_size": 3})
            print(res.content[0].text)   # markdown 表格文本

asyncio.run(main())
```
