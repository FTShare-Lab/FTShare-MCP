# 新闻语义搜索（MCP 工具 `ft_semantic_search_news_handler`）

> **MCP 工具**：`ft_semantic_search_news_handler`（category: `大模型语料`）。返回 **markdown 表格文本**（非结构化 JSON）。输入参数 / 输出参数 / 数据样例对照 [ftshare-doc](../../ftshare-doc/api-doc) 原接口。

- 描述：语义搜索新闻接口。根据搜索文字 `query` 转调外部语义检索服务，返回相关新闻列表。支持按 `year`（年份）或 `start_time` / `end_time`（时间区间）限定搜索范围，时间区间需在同一 3 天窗口内（校验 `validate_datetime_range_3_days`）。响应为 `SearchResult` 列表，含新闻 ID、来源站点、文章链接、发布/抓取时间、标题、媒体、摘要、正文、匹配分数等。
- 数据范围：依赖外部语义检索服务的数据范围（实时检索，无固定时间维度）
- 单次限量：返回条数由 `limit` 控制，默认由服务端决定
- 提示：
  - `query` 必填。
  - 同时传 `start_time` / `end_time` 时，区间跨度不得超过 3 天。
  - 接口转调外部服务，开放路由，可能受上游可用性与延迟影响。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| query | string | Y | 搜索文字 |
| limit | int | N | 返回条数，默认由服务端决定 |
| year | int | N | 年份，限定搜索范围 |
| start_time | datetime | N | 起始时间（带时区），与 end_time 同传时区间不超过 3 天 |
| end_time | datetime | N | 结束时间（带时区），与 start_time 同传时区间不超过 3 天 |

## 输出参数

SearchResult：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| news_id | uint64 | Y | 新闻 ID |
| source_site | string | Y | 来源站点 |
| article_url | string | N | 文章链接 |
| publish_time | datetime | N | 发布时间 |
| fetch_time | datetime | Y | 抓取时间 |
| title | string | N | 标题 |
| media_name | string | N | 媒体名称 |
| summary | string | N | 摘要 |
| content | string | Y | 正文内容 |
| is_truncated | int8 | Y | 是否被截断（0/1） |
| is_reviewed | int8 | Y | 是否已审核（0/1） |
| score | float64 | Y | 搜索匹配分数 |

## 调用方法（MCP）

> MCP 工具名 `ft_semantic_search_news_handler`。MCP Streamable HTTP 要求**先 initialize 拿 `Mcp-Session-Id`，再 `tools/call`**，后续请求带该 header。返回 **markdown 表格文本**。

**curl**：

```bash
SID=$(curl -sS -m 10 -D - -o /dev/null -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"t","version":"1"}}}' \
  | grep -i 'mcp-session-id:' | awk '{print $2}' | tr -d '')

curl -sS -m 10 -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -H "Mcp-Session-Id: $SID" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ft_semantic_search_news_handler","arguments":{"query": "宁德时代"}}}'
```

**Python（`mcp` SDK，自动握手管理 session）**：

```python
import asyncio
from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client

async def main():
    async with streamablehttp_client('<MCP_BASE_URL>') as (r, w, _):
        async with ClientSession(r, w) as s:
            await s.initialize()
            res = await s.call_tool('ft_semantic_search_news_handler', {"query": "宁德时代"})
            print(res.content[0].text)   # markdown 表格文本

asyncio.run(main())
```
