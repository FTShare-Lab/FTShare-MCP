# 股东大会（MCP 工具 `ft_shareholders_meeting`）

> **MCP 工具**：`ft_shareholders_meeting`（category: `大模型语料`）。返回 **markdown 表格文本**（非结构化 JSON）。输入参数 / 输出参数 / 数据样例对照 [ftshare-doc](../../ftshare-doc/api-doc) 原接口。

- 描述：查询 A 股上市公司股东大会通知信息，返回每场股东大会的会议名称、股权登记日、现场/网络投票起止日、决议公告日、通知日期及审议议案等。数据来源于东方财富（eastmoney）股东大会公告，底层通过 `load_shareholders_meeting` 全量加载后分页返回。响应为 `PaginatedApiResponse` 信封（code/message/data），data 为分页对象。
- 数据范围：以服务端返回为准
- 单次限量：分页返回，默认 `page=1`，`page_size` 由后端定
- 提示：
  - 无请求过滤参数，全量分页返回所有股东大会通知。
  - 多数字段为 `Option<String>`，缺失字段返回 null。
  - `start_adjust_date` 为开始调整日期，`equity_record_date` 为股权登记日。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| page | int | N | 页码，默认 1 |
| page_size | int | N | 每页条数 |

## 输出参数

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| code | int | Y | 状态码：0 成功 |
| message | string | Y | 状态信息 |
| data | object | Y | 分页对象（见下） |

data ：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| pageNum | int | Y | 当前页码 |
| pageSize | int | Y | 每页条数 |
| total | int | Y | 总记录数 |
| pages | int | Y | 总页数 |
| records | array[ShareholdersMeetingItem] | Y | 股东大会列表 |

ShareholdersMeetingItem：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| security_code | string | Y | 证券代码 |
| security_name_abbr | string | Y | 证券简称 |
| meeting_title | string | Y | 会议名称（如「2026年第2次临时股东大会」） |
| start_adjust_date | string | Y | 开始调整日期 |
| equity_record_date | string | Y | 股权登记日 |
| onsite_record_date | string | Y | 现场登记日 |
| web_start_date | string | Y | 网络投票开始日 |
| web_end_date | string | Y | 网络投票结束日 |
| decision_notice_date | string | Y | 决议公告日 |
| notice_date | string | Y | 通知日期 |
| serial_num | string | Y | 序号 |
| proposal | string | Y | 审议议案 |

## 调用方法（MCP）

> MCP 工具名 `ft_shareholders_meeting`。MCP Streamable HTTP 要求**先 initialize 拿 `Mcp-Session-Id`，再 `tools/call`**，后续请求带该 header。返回 **markdown 表格文本**。

**curl**：

```bash
SID=$(curl -sS -m 10 -D - -o /dev/null -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"t","version":"1"}}}' \
  | grep -i 'mcp-session-id:' | awk '{print $2}' | tr -d '')

curl -sS -m 10 -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -H "Mcp-Session-Id: $SID" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ft_shareholders_meeting","arguments":{}}}'
```

**Python（`mcp` SDK，自动握手管理 session）**：

```python
import asyncio
from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client

async def main():
    async with streamablehttp_client('<MCP_BASE_URL>') as (r, w, _):
        async with ClientSession(r, w) as s:
            await s.initialize()
            res = await s.call_tool('ft_shareholders_meeting', {})
            print(res.content[0].text)   # markdown 表格文本

asyncio.run(main())
```
