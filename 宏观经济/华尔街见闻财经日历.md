# 华尔街见闻财经日历（MCP 工具 `ft_wallstreetcn_financial_calendar`）

> **MCP 工具**：`ft_wallstreetcn_financial_calendar`（category: `宏观经济/国内宏观`）。返回 **markdown 表格文本**（非结构化 JSON）。输入参数 / 输出参数 / 数据样例对照 [ftshare-doc](../../ftshare-doc/api-doc) 原接口。

- 描述：查询华尔街见闻（wallstreetcn）财经日历，按日期范围返回该区间内的经济事件。每条含事件日期、事件时间、地区、事件名称、重要性、实际值/预测值/前值及原文链接。数据来源：华尔街见闻。响应为分页信封 `PaginatedApiResponse`（code/message/data），`data` 为内存分页结果（pageNum/pageSize/total/pages/records）。底层对 `start_date`/`end_date` 做 3 天范围校验（`validate_date_str_range_3_days`）。
- 数据范围：按日期范围，单次请求窗 ≤ 3 天
- 单次限量：分页返回，`page_size` 默认 50、最大 200；`page` 默认 1
- 提示：
  - `start_date` / `end_date` 必填，两者间隔不得超过 3 天，否则返回 code=400 错误。
  - 数据来源：华尔街见闻（路径中保留 `wallstreetcn` 来源标识）。
  - 响应为 code/message/data 信封，data 含 records 数组与分页字段。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| start_date | string | Y | 起始日期 |
| end_date | string | Y | 结束日期（与 start_date 间隔 ≤ 3 天） |
| page | int | N | 页码，从 1 开始，默认 1 |
| page_size | int | N | 每页条数，默认 50，最大 200 |

## 输出参数

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| code | int | Y | 状态码：0 成功，400 参数/范围错误 |
| message | string | Y | 状态信息 |
| data | object | Y | 分页对象（见下） |

data ：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| pageNum | int | Y | 当前页码 |
| pageSize | int | Y | 每页条数 |
| total | int | Y | 总记录数 |
| pages | int | Y | 总页数 |
| records | array[WallstreetcnCalendarItem] | Y | 财经日历事件列表 |

WallstreetcnCalendarItem：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| stat_date | string | Y | 事件日期 |
| tab_type | string | Y | 事件分类标签 |
| event_time | string | Y | 事件时间 |
| region | string | Y | 地区 |
| event | string | Y | 事件名称 |
| importance | string | Y | 重要性 |
| actual_value | string | N | 实际值 |
| forecast_value | string | N | 预测值 |
| previous_value | string | N | 前值 |
| link | string | N | 原文链接 |

## 调用方法（MCP）

> MCP 工具名 `ft_wallstreetcn_financial_calendar`。MCP Streamable HTTP 要求**先 initialize 拿 `Mcp-Session-Id`，再 `tools/call`**，后续请求带该 header。返回 **markdown 表格文本**。

**curl**：

```bash
SID=$(curl -sS -m 10 -D - -o /dev/null -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"t","version":"1"}}}' \
  | grep -i 'mcp-session-id:' | awk '{print $2}' | tr -d '')

curl -sS -m 10 -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -H "Mcp-Session-Id: $SID" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ft_wallstreetcn_financial_calendar","arguments":{"start_date": "2026-06-20", "end_date": "2026-06-23"}}}'
```

**Python（`mcp` SDK，自动握手管理 session）**：

```python
import asyncio
from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client

async def main():
    async with streamablehttp_client('<MCP_BASE_URL>') as (r, w, _):
        async with ClientSession(r, w) as s:
            await s.initialize()
            res = await s.call_tool('ft_wallstreetcn_financial_calendar', {"start_date": "2026-06-20", "end_date": "2026-06-23"})
            print(res.content[0].text)   # markdown 表格文本

asyncio.run(main())
```
