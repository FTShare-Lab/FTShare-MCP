# 百度财经日历（MCP 工具 `ft_baidu_financial_calendar`）

> **MCP 工具**：`ft_baidu_financial_calendar`（category: `宏观经济/国内宏观`）。返回 **markdown 表格文本**（非结构化 JSON）。输入参数 / 输出参数 / 数据样例对照 [ftshare-doc](../../ftshare-doc/api-doc) 原接口。

- 描述：查询百度财经日历，按日期范围返回该区间内的财经事件，覆盖经济数据、新股 IPO、财报披露时间、交易提醒四类。可通过 `category` 筛选单类，不传返回全部。响应为分页信封 `PaginatedApiResponse`，`records` 为带类型标签的枚举（`category` 字段区分四类，各类字段同级扁平）。底层对 `start_date`/`end_date` 做 3 天范围校验，并按 (stat_date, time, title/code) 升序排序。数据来源：百度。返回结果先全量加载再内存分页。
- 数据范围：按日期范围，单次请求窗 ≤ 3 天
- 单次限量：分页返回，`page_size` 默认 50、最大 200；`page` 默认 1
- 提示：
  - `start_date` / `end_date` 必填，间隔 ≤ 3 天，否则返回 code=400。
  - `category` 可选，取值 `economic`/`ipo`/`report_time`/`trade_reminder`；非法值返回 code=400。
  - 数据来源：百度（路径中保留 `baidu` 来源标识）。
  - records 每行带 `category` 标签，四类条目字段不同，下表合并列出（缺失字段为 null）。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| start_date | string | Y | 起始日期 |
| end_date | string | Y | 结束日期（与 start_date 间隔 ≤ 3 天） |
| category | string | N | 筛选大类：`economic` / `ipo` / `report_time` / `trade_reminder`；不传返回全部 |
| page | int | N | 页码，从 1 开始，默认 1 |
| page_size | int | N | 每页条数，默认 50，最大 200 |

## 调用方法（MCP）

> MCP 工具名 `ft_baidu_financial_calendar`。MCP Streamable HTTP 要求**先 initialize 拿 `Mcp-Session-Id`，再 `tools/call`**，后续请求带该 header。返回 **markdown 表格文本**。

**curl**：

```bash
SID=$(curl -sS -m 10 -D - -o /dev/null -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"t","version":"1"}}}' \
  | grep -i 'mcp-session-id:' | awk '{print $2}' | tr -d '')

curl -sS -m 10 -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -H "Mcp-Session-Id: $SID" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ft_baidu_financial_calendar","arguments":{"start_date": "2026-06-20", "end_date": "2026-06-23"}}}'
```

**Python（`mcp` SDK，自动握手管理 session）**：

```python
import asyncio
from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client

async def main():
    async with streamablehttp_client('<MCP_BASE_URL>') as (r, w, _):
        async with ClientSession(r, w) as s:
            await s.initialize()
            res = await s.call_tool('ft_baidu_financial_calendar', {"start_date": "2026-06-20", "end_date": "2026-06-23"})
            print(res.content[0].text)   # markdown 表格文本

asyncio.run(main())
```
