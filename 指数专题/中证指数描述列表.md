# 中证指数描述列表（MCP 工具 `ft_index_description_list_handler`）

> **MCP 工具**：`ft_index_description_list_handler`（category: `指数专题/指数基本信息`）。返回 **markdown 表格文本**（非结构化 JSON）。输入参数 / 输出参数 / 数据样例对照 [ftshare-doc](../../ftshare-doc/api-doc) 原接口。

- 描述：分页获取中证系列指数（中证指数有限公司发布）的描述信息，含指数代码、来源机构（如「中证指数」）、指数名称、指数简介（编制说明文字）。返回**分页结构**：`total` 为指数总数，`index_descriptions` 为当页列表。每项含 `url_hash`，可用其调用 `index_description/{url_hash}` 下载该指数的 PDF 编制文件。底层转调外部爬虫服务。
- 数据范围：无时间维度，最新快照（共约 825 只中证系列指数）
- 单次限量：分页，`page_size` 默认 20、最大 100
- 提示：
  - 路径含 `index` 子前缀：`/market/data/index/index_description`。
  - `index_orig` 标识发布机构（中证指数等），与 `index_name` 不同。
  - `url_hash` 可用于下载对应指数编制细则 PDF（走 `index_description/{url_hash}` 文件接口）。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| page | int | N | 页码，默认 1 |
| page_size | int | N | 每页条数，默认 20，最大 100 |

## 输出参数

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| total | int | Y | 指数总数 |
| page | int | Y | 当前页码 |
| page_size | int | Y | 当前每页条数 |
| index_descriptions | array | Y | 当页指数描述列表 |

index_descriptions 每项（IndexDescriptionInfo）：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| index_code | string | Y | 指数代码，如 000001 |
| index_orig | string | Y | 发布机构/来源，如 中证指数 |
| index_name | string | Y | 指数名称，如 上证综合指数 |
| index_intro | string | Y | 指数简介/编制说明 |
| url_hash | string | Y | 文件下载 hash，用于下载 PDF 编制文件 |

## 调用方法（MCP）

> MCP 工具名 `ft_index_description_list_handler`。MCP Streamable HTTP 要求**先 initialize 拿 `Mcp-Session-Id`，再 `tools/call`**，后续请求带该 header。返回 **markdown 表格文本**。

**curl**：

```bash
SID=$(curl -sS -m 10 -D - -o /dev/null -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"t","version":"1"}}}' \
  | grep -i 'mcp-session-id:' | awk '{print $2}' | tr -d '')

curl -sS -m 10 -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -H "Mcp-Session-Id: $SID" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ft_index_description_list_handler","arguments":{}}}'
```

**Python（`mcp` SDK，自动握手管理 session）**：

```python
import asyncio
from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client

async def main():
    async with streamablehttp_client('<MCP_BASE_URL>') as (r, w, _):
        async with ClientSession(r, w) as s:
            await s.initialize()
            res = await s.call_tool('ft_index_description_list_handler', {})
            print(res.content[0].text)   # markdown 表格文本

asyncio.run(main())
```
