# 指数基础信息（MCP 工具 `ft_index_description_all`）

> **MCP 工具**：`ft_index_description_all`（category: `指数专题/指数基本信息`）。返回 **markdown 表格文本**（非结构化 JSON）。输入参数 / 输出参数 / 数据样例对照 [ftshare-doc](../../ftshare-doc/api-doc) 原接口。

- 描述：获取全部指数的基础信息（全称、简称、市净率 PB、市盈率 TTM），返回**数组**（非分页）。覆盖沪深京三市全部指数，按 `symbol` 升序排列。每项含指数代码（如 `000001.XSHG`）、全称（如「上证综合指数」）、简称（如「上证指数」）。市净率/市盈率为最新快照值，部分退市或等权指数（如 `000070.XSHG` 能源等权）该字段为 `null`。
- 数据范围：无时间维度，实时/最新快照
- 单次限量：无入参、无分页，一次返回全部指数（约 700+ 条）
- 提示：
  - 无入参，直接 GET 即可。
  - `symbol` 为标的代码，含交易所后缀：上交所 `.XSHG`、深交所 `.XSHE`、北交所 `.BJSE`。
  - `pb` / `pe_ttm` 为最新估值快照，退市/无估值指数为 `null`。

## 输入参数

无（无需传参）。

## 输出参数

数组，每项（IndexDescriptionResponse）：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| symbol | string | Y | 指数代码，含交易所后缀，如 000001.XSHG |
| full_name | string | Y | 指数全称，如 上证综合指数 |
| name | string | Y | 指数简称，如 上证指数 |
| pb | number/null | Y | 市净率（最新快照），无估值时为 null |
| pe_ttm | number/null | Y | 市盈率 TTM（最新快照），无估值时为 null |

## 调用方法（MCP）

> MCP 工具名 `ft_index_description_all`。MCP Streamable HTTP 要求**先 initialize 拿 `Mcp-Session-Id`，再 `tools/call`**，后续请求带该 header。返回 **markdown 表格文本**。

**curl**：

```bash
SID=$(curl -sS -m 10 -D - -o /dev/null -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"t","version":"1"}}}' \
  | grep -i 'mcp-session-id:' | awk '{print $2}' | tr -d '')

curl -sS -m 10 -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -H "Mcp-Session-Id: $SID" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ft_index_description_all","arguments":{}}}'
```

**Python（`mcp` SDK，自动握手管理 session）**：

```python
import asyncio
from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client

async def main():
    async with streamablehttp_client('<MCP_BASE_URL>') as (r, w, _):
        async with ClientSession(r, w) as s:
            await s.initialize()
            res = await s.call_tool('ft_index_description_all', {})
            print(res.content[0].text)   # markdown 表格文本

asyncio.run(main())
```
