# 指数权重列表（MCP 工具 `ft_index_weight_list_handler`）

> **MCP 工具**：`ft_index_weight_list_handler`（category: `指数专题/指数成分和权重`）。返回 **markdown 表格文本**（非结构化 JSON）。输入参数 / 输出参数 / 数据样例对照 [ftshare-doc](../../ftshare-doc/api-doc) 原接口。

- 描述：分页获取单只中证系列指数某一期的**成份股权重明细**。返回**分页结构**，每项为一只成份股（`component_code` + `component_name`）在该期（`date`）的权重（`weight`，小数字符串）。典型大盘指数（如沪深300）单期约 300 只成份，总指数 1232 条（跨多期）。`url_hash` 可用于下载整份权重 xlsx 文件。`index_code` 必填，`date` 可选（不传则取该指数最新期）。底层转调外部爬虫服务。
- 数据范围：无时间维度，最新若干期快照（沪深300 当前可取到 2026-05-29）
- 单次限量：分页，`page_size` 默认 20、最大 100
- 提示：
  - 路径含 `index` 子前缀：`/market/data/index/index_weight`。
  - `index_code` **必填**，为空会被拒（400）。
  - `date` 可选；建议先从 `index_weight_summary` 拿到可用 `date` 再按期查。
  - `weight` 为字符串形式的小数权重（如 `0.0609`），非百分比。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| index_code | string | Y | 指数代码，如 000300（沪深300），为空会被拒 |
| date | string | N | 权重采集日期 YYYYMMDD（如 20260529），不传取最新期 |
| page | int | N | 页码，默认 1 |
| page_size | int | N | 每页条数，默认 20，最大 100 |

## 输出参数

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| total | int | Y | 命中总条数（跨多期时为所有期成份累计） |
| page | int | Y | 当前页码 |
| page_size | int | Y | 当前每页条数 |
| index_weights | array | Y | 当页成份权重明细列表 |

index_weights 每项（IndexWeightInfo）：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| date | string | Y | 权重采集日期 YYYY-MM-DD |
| index_code | string | Y | 指数代码，如 000300 |
| component_code | string | Y | 成份股代码，如 601398 |
| component_name | string | Y | 成份股名称，如 工商银行 |
| weight | string | Y | 权重（小数字符串，如 0.0609） |
| url_hash | string | Y | 文件下载 hash，用于下载该期 xlsx 权重文件 |

## 调用方法（MCP）

> MCP 工具名 `ft_index_weight_list_handler`。MCP Streamable HTTP 要求**先 initialize 拿 `Mcp-Session-Id`，再 `tools/call`**，后续请求带该 header。返回 **markdown 表格文本**。

**curl**：

```bash
SID=$(curl -sS -m 10 -D - -o /dev/null -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"t","version":"1"}}}' \
  | grep -i 'mcp-session-id:' | awk '{print $2}' | tr -d '')

curl -sS -m 10 -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -H "Mcp-Session-Id: $SID" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ft_index_weight_list_handler","arguments":{"index_code": "000300"}}}'
```

**Python（`mcp` SDK，自动握手管理 session）**：

```python
import asyncio
from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client

async def main():
    async with streamablehttp_client('<MCP_BASE_URL>') as (r, w, _):
        async with ClientSession(r, w) as s:
            await s.initialize()
            res = await s.call_tool('ft_index_weight_list_handler', {"index_code": "000300"})
            print(res.content[0].text)   # markdown 表格文本

asyncio.run(main())
```
