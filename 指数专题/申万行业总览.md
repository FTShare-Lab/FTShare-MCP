# 申万行业总览（MCP 工具 `ft_sw_industry_overview`）

> **MCP 工具**：`ft_sw_industry_overview`（category: `指数专题/申万行业`）。返回 **markdown 表格文本**（非结构化 JSON）。输入参数 / 输出参数 / 数据样例对照 [ftshare-doc](../../ftshare-doc/api-doc) 原接口。

- 描述：按交易日查询申万行业**全部层级**（一/二/三级行业）的估值概况快照，含成份股数、PE（LYR/TTM）及历史分位、PB 及分位、股息率（dv_ratio）及分位。返回 `PaginatedApiResponse`（自定义 `{code,message,data:{pageNum,pageSize,total,pages,records}}` 形态，字段为 camelCase）。单日全层级约 498 条。
- 数据范围：2026-05-28 至今（跨多交易日探测，2026-05-28 起稳定有数据，更早日期返回空）
- 单次限量：由 page/page_size 分页；默认返回全部命中
- 提示：
  - `date` 必填，格式 **YYYYMMDD**（带 `-` 会报 invalid date format）。
  - `level` 可选，传 1/2/3 过滤行业层级；不传返回全部层级。
  - 返回为 `{code,message,data}` 包裹的分页结构，`records` 为行数组，非顶层 items/total_items。
  - 估值字段均为字符串。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| date | string | Y | 交易日，格式 YYYYMMDD |
| level | int | N | 行业层级：1/2/3，不传返回全部 |
| page | int | N | 页码，从 1 开始 |
| page_size | int | N | 每页数量 |

## 输出参数

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| code | int | Y | 0=成功，非 0=错误 |
| message | string | Y | 状态说明 |
| data.pageNum | int | Y | 当前页码 |
| data.pageSize | int | Y | 每页数量 |
| data.total | int | Y | 总记录数 |
| data.pages | int | Y | 总页数 |
| data.records | array | Y | 行业概况列表 |

SwIndustryOverviewItem：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| industryCode | string | Y | 申万行业代码，如 801010.SI |
| industryName | string | Y | 行业名称（中文） |
| level | int | Y | 行业层级 1/2/3 |
| parentIndustryName | string | N | 父级行业名称 |
| constituentCount | string | N | 成份股数量 |
| tradeDate | string | N | 交易日期 |
| lyrPe | string | N | 静态 PE（LYR） |
| lyrPePercentile | string | N | LYR PE 历史分位（%） |
| ttmPe | string | N | TTM PE |
| ttmPePercentile | string | N | TTM PE 历史分位（%） |
| pb | string | N | PB |
| pbPercentile | string | N | PB 历史分位（%） |
| dvRatio | string | N | 股息率（%） |
| dvRatioPercentile | string | N | 股息率历史分位（%） |

## 调用方法（MCP）

> MCP 工具名 `ft_sw_industry_overview`。MCP Streamable HTTP 要求**先 initialize 拿 `Mcp-Session-Id`，再 `tools/call`**，后续请求带该 header。返回 **markdown 表格文本**。

**curl**：

```bash
SID=$(curl -sS -m 10 -D - -o /dev/null -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"t","version":"1"}}}' \
  | grep -i 'mcp-session-id:' | awk '{print $2}' | tr -d '')

curl -sS -m 10 -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -H "Mcp-Session-Id: $SID" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ft_sw_industry_overview","arguments":{"date": "20260624"}}}'
```

**Python（`mcp` SDK，自动握手管理 session）**：

```python
import asyncio
from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client

async def main():
    async with streamablehttp_client('<MCP_BASE_URL>') as (r, w, _):
        async with ClientSession(r, w) as s:
            await s.initialize()
            res = await s.call_tool('ft_sw_industry_overview', {"date": "20260624"})
            print(res.content[0].text)   # markdown 表格文本

asyncio.run(main())
```
