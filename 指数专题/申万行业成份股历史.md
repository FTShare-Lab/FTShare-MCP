# 申万行业成份股历史（MCP 工具 `ft_sw_industry_constituent_history`）

> **MCP 工具**：`ft_sw_industry_constituent_history`（category: `指数专题/申万行业`）。返回 **markdown 表格文本**（非结构化 JSON）。输入参数 / 输出参数 / 数据样例对照 [ftshare-doc](../../ftshare-doc/api-doc) 原接口。

- 描述：查询单个申万行业的**成份股调入调出历史**，返回该行业（含一/二/三级行业归属）下的全部成份股及其调入日期（inDate）、调出日期（outDate）。返回 `ApiResponse` 形态（`{code,message,data:{total,items}}`，字段 camelCase）。典型一级行业约 300 只成份股。
- 数据范围：成份股调入日期可追溯至 2012 年（如 002688 农林牧渔成份 2012-01-05 调入），反映申万分类历史变更
- 单次限量：无分页，一次返回该行业全部成份股历史
- 提示：
  - `industry_code` 必填，带 `.SI` 后缀，如 801010.SI。
  - 返回非分页，`data.items` 为全部记录，`data.total` 为数量。
  - `inDate`/`outDate` 可为 null（仍在职）。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| industry_code | string | Y | 行业代码，带 .SI 后缀，如 801010.SI |

## 输出参数

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| code | int | Y | 0=成功，非 0=错误 |
| message | string | Y | 状态说明 |
| data.total | int | Y | 成份股记录总数 |
| data.items | array | Y | 成份股历史列表 |

SwIndustryConstituentHistoryItem：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| stockCode | string | Y | 股票代码（6 位） |
| stockName | string | Y | 股票名称 |
| inDate | string | N | 调入日期 |
| outDate | string | N | 调出日期（在职为 null） |
| swLevel1Code | string | Y | 一级行业代码 |
| swLevel1Name | string | Y | 一级行业名称 |
| swLevel2Code | string | N | 二级行业代码 |
| swLevel2Name | string | N | 二级行业名称 |
| swLevel3Code | string | N | 三级行业代码 |
| swLevel3Name | string | N | 三级行业名称 |

## 调用方法（MCP）

> MCP 工具名 `ft_sw_industry_constituent_history`。MCP Streamable HTTP 要求**先 initialize 拿 `Mcp-Session-Id`，再 `tools/call`**，后续请求带该 header。返回 **markdown 表格文本**。

**curl**：

```bash
SID=$(curl -sS -m 10 -D - -o /dev/null -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"t","version":"1"}}}' \
  | grep -i 'mcp-session-id:' | awk '{print $2}' | tr -d '')

curl -sS -m 10 -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -H "Mcp-Session-Id: $SID" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ft_sw_industry_constituent_history","arguments":{"industry_code": "801010.SI"}}}'
```

**Python（`mcp` SDK，自动握手管理 session）**：

```python
import asyncio
from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client

async def main():
    async with streamablehttp_client('<MCP_BASE_URL>') as (r, w, _):
        async with ClientSession(r, w) as s:
            await s.initialize()
            res = await s.call_tool('ft_sw_industry_constituent_history', {"industry_code": "801010.SI"})
            print(res.content[0].text)   # markdown 表格文本

asyncio.run(main())
```
