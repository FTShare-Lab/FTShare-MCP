# 申万行业日度指标（MCP 工具 `ft_sw_industry_daily_metrics`）

> **MCP 工具**：`ft_sw_industry_daily_metrics`（category: `指数专题/申万行业`）。返回 **markdown 表格文本**（非结构化 JSON）。输入参数 / 输出参数 / 数据样例对照 [ftshare-doc](../../ftshare-doc/api-doc) 原接口。

- 描述：查询单个申万行业的**日频估值/资金/量价**指标时间序列，含 PE（LYR/TTM 及分位/中位数）、PB、股息率、换手率、总市值、收盘价、破净比例、指数收盘、成交额及拥挤度等约 40 项字段（字段全部为字符串）。按行业层级（1/2/3）分表查询，单行业约 1200+ 个交易日。
- 数据范围：2021-05-31 至今（跨 801010.SI/801030.SI/801080.SI 等多行业探测，最早均为 2021-05-31）
- 单次限量：无 3 天跨度限制，由 page/page_size 分页
- 提示：
  - `level` 必填（1/2/3），`start_date`/`end_date`/`industry_code` 必填。
  - 日期格式 **YYYYMMDD**；`industry_code` 必须带 `.SI` 后缀（如 801010.SI），不带会查不到数据。
  - 返回 `{code,message,data:{pageNum,pageSize,total,pages,records}}` 形态，字段 camelCase、全部为字符串。
  - 字段较多，下方输出参数表列核心列，完整字段见协议结构 `SwIndustryDailyMetricsItem`。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| level | int | Y | 行业层级：1/2/3 |
| start_date | string | Y | 起始日期，YYYYMMDD |
| end_date | string | Y | 截止日期，YYYYMMDD |
| industry_code | string | Y | 行业代码，带 .SI 后缀，如 801010.SI |
| page | int | N | 页码，从 1 开始 |
| page_size | int | N | 每页数量 |

## 输出参数

SwIndustryDailyMetricsItem：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| industryCode | string | Y | 行业代码 |
| industryName | string | Y | 行业名称 |
| tradeDate | string | Y | 交易日 |
| lyrPe / lyrPeQuantile | string | Y | 静态 PE 及分位 |
| ttmPe / ttmPeQuantile | string | Y | TTM PE 及分位 |
| pb / pbQuantile | string | Y | PB 及分位 |
| dvRatio / dvRatioQuantile | string | Y | 股息率及分位 |
| turnoverRate | string | Y | 换手率（%） |
| totalMv | string | Y | 总市值 |
| closePrice | string | Y | 收盘价 |
| indexClose | string | Y | 行业指数收盘 |
| amount | string | Y | 成交额 |
| totalCount | string | Y | 成份股数量 |
| belowNetAssetCount | string | Y | 破净成份股数量 |

其余字段（`add*` 加权指标、`middle*` 中位数指标、`dvTtm*`、`turnoverRateF*`、`amountCongestion*` 等）见协议结构。

## 调用方法（MCP）

> MCP 工具名 `ft_sw_industry_daily_metrics`。MCP Streamable HTTP 要求**先 initialize 拿 `Mcp-Session-Id`，再 `tools/call`**，后续请求带该 header。返回 **markdown 表格文本**。

**curl**：

```bash
SID=$(curl -sS -m 10 -D - -o /dev/null -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"t","version":"1"}}}' \
  | grep -i 'mcp-session-id:' | awk '{print $2}' | tr -d '')

curl -sS -m 10 -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -H "Mcp-Session-Id: $SID" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ft_sw_industry_daily_metrics","arguments":{"level": 1, "start_date": "2026-06-20", "end_date": "2026-06-23", "industry_code": "801010.SI"}}}'
```

**Python（`mcp` SDK，自动握手管理 session）**：

```python
import asyncio
from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client

async def main():
    async with streamablehttp_client('<MCP_BASE_URL>') as (r, w, _):
        async with ClientSession(r, w) as s:
            await s.initialize()
            res = await s.call_tool('ft_sw_industry_daily_metrics', {"level": 1, "start_date": "2026-06-20", "end_date": "2026-06-23", "industry_code": "801010.SI"})
            print(res.content[0].text)   # markdown 表格文本

asyncio.run(main())
```
