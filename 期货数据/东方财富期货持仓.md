# 东方财富期货持仓（MCP 工具 `ft_get_eastmoney_futures_position`）

> **MCP 工具**：`ft_get_eastmoney_futures_position`（category: `期货数据`）。返回 **markdown 表格文本**（非结构化 JSON）。输入参数 / 输出参数 / 数据样例对照 [ftshare-doc](../../ftshare-doc/api-doc) 原接口。

- 描述：获取东方财富**期货每日成交持仓排行榜**数据，按合约/会员维度给出成交量、多头、空头、净多头、净空头的排名与持仓量及变化，含多头增/减仓、空头增/减仓排名，以及结算价、多头/空头平均价。数据来源：东方财富，底表为 ClickHouse `basedata.em_futures_daily_position`（多数字段 Nullable）。支持按交易所、品种、合约、会员、交易日或日期区间过滤。响应为 `PaginatedApiResponse` 信封（code/message/data）。
- 数据范围：持仓类，最新一期或当日快照（历史序列下限以服务端返回为准）
- 单次限量：分页返回，默认 `page=1`、`page_size=50`、最大 `page_size=200`
- 提示：
  - `exchange` 取值：shfe（上期所）/ dce（大商所）/ czce（郑商所）/ cffex（中金所）/ ine（能源中心）/ gfe（广期所）。
  - 每个合约每个交易日下会有多会员多行，总量较大，建议按合约+日期精准过滤。
  - `trade_date` 与 `start_date`/`end_date` 互斥。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| exchange | string | N | 交易所代码：shfe / dce / czce / cffex / ine / gfe |
| variety_code | string | N | 品种代码，如 cu / au / al / IF |
| contract_code | string | N | 合约代码，如 CU2607 / AU2608 |
| trade_date | string | N | 交易日 YYYYMMDD；与 start_date/end_date 互斥 |
| start_date | string | N | 区间起始日 YYYYMMDD；需与 end_date 同时提供 |
| end_date | string | N | 区间结束日 YYYYMMDD；需与 start_date 同时提供 |
| member_name_abbr | string | N | 会员简称 |
| page | int | N | 页码，从 1 开始，默认 1 |
| page_size | int | N | 每页条数，默认 50，最大 200 |

## 调用方法（MCP）

> MCP 工具名 `ft_get_eastmoney_futures_position`。MCP Streamable HTTP 要求**先 initialize 拿 `Mcp-Session-Id`，再 `tools/call`**，后续请求带该 header。返回 **markdown 表格文本**。

**curl**：

```bash
SID=$(curl -sS -m 10 -D - -o /dev/null -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"t","version":"1"}}}' \
  | grep -i 'mcp-session-id:' | awk '{print $2}' | tr -d '')

curl -sS -m 10 -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -H "Mcp-Session-Id: $SID" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ft_get_eastmoney_futures_position","arguments":{}}}'
```

**Python（`mcp` SDK，自动握手管理 session）**：

```python
import asyncio
from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client

async def main():
    async with streamablehttp_client('<MCP_BASE_URL>') as (r, w, _):
        async with ClientSession(r, w) as s:
            await s.initialize()
            res = await s.call_tool('ft_get_eastmoney_futures_position', {})
            print(res.content[0].text)   # markdown 表格文本

asyncio.run(main())
```
