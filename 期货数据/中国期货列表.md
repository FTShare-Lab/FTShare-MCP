# 中国期货列表（MCP 工具 `ft_get_china_futures_lists_handler`）

> **MCP 工具**：`ft_get_china_futures_lists_handler`（category: `期货数据`）。返回 **markdown 表格文本**（非结构化 JSON）。输入参数 / 输出参数 / 数据样例对照 [ftshare-doc](../../ftshare-doc/api-doc) 原接口。

- 描述：查询指定交易日的期货合约列表（合约代码、交易所、品种、中文名称、合约类型）。不传 `trade_date` 默认前一交易日（CST）。返回 `ChinaFuturesListsResponse`，由 `get_china_futures_lists` 读取。
- 数据范围：最新快照（按交易日的合约清单，无时间序列）
- 单次限量：单交易日全量合约列表，无分页
- 提示：
  - `trade_date` 不传则使用前一交易日（CST）。
  - 不按 `future_type` 过滤，原样带出合约类型。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| trade_date | int | N | 交易日 YYYYMMDD；不传则使用前一交易日（CST） |

## 输出参数

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| trade_date | int | Y | 实际使用的交易日（与请求或默认前一交易日一致） |
| items | array[ChinaFuturesListsItem] | Y | 期货列表 |

ChinaFuturesListsItem：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| symbol | string | Y | 合约代码 |
| exchange | string | Y | 交易所 |
| product | string | Y | 品种 |
| symbol_cn_name | string | Y | 合约中文名称 |
| future_type | int | N | 合约类型：1 主力 / 2 真实 / 3 连续；源数据缺失为 null |

## 调用方法（MCP）

> MCP 工具名 `ft_get_china_futures_lists_handler`。MCP Streamable HTTP 要求**先 initialize 拿 `Mcp-Session-Id`，再 `tools/call`**，后续请求带该 header。返回 **markdown 表格文本**。

**curl**：

```bash
SID=$(curl -sS -m 10 -D - -o /dev/null -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"t","version":"1"}}}' \
  | grep -i 'mcp-session-id:' | awk '{print $2}' | tr -d '')

curl -sS -m 10 -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -H "Mcp-Session-Id: $SID" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ft_get_china_futures_lists_handler","arguments":{}}}'
```

**Python（`mcp` SDK，自动握手管理 session）**：

```python
import asyncio
from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client

async def main():
    async with streamablehttp_client('<MCP_BASE_URL>') as (r, w, _):
        async with ClientSession(r, w) as s:
            await s.initialize()
            res = await s.call_tool('ft_get_china_futures_lists_handler', {})
            print(res.content[0].text)   # markdown 表格文本

asyncio.run(main())
```
