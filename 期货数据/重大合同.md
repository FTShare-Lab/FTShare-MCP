# 重大合同（MCP 工具 `ft_major_contract`）

> **MCP 工具**：`ft_major_contract`（category: `期货数据`）。返回 **markdown 表格文本**（非结构化 JSON）。输入参数 / 输出参数 / 数据样例对照 [ftshare-doc](../../ftshare-doc/api-doc) 原接口。

- 描述：按签约日期区间查询 A 股上市公司披露的重大合同，返回合同双方、合同类型/名称、合同金额、占上年营收比例、签约日/披露日等。数据来源于东方财富（eastmoney）重大合同数据。`start_date` / `end_date` 必填且区间跨度受 `validate_date_str_range_3_days` 约束（最大 3 天）。响应为 `ApiResponse` 信封，data 含 `total` 与 `items`。
- 数据范围：以服务端返回为准
- 单次限量：查询区间最大跨度 3 天（超出返回 400）
- 提示：
  - `start_date` / `end_date` 必填，日期区间跨度不得超过 3 天。
  - `amounts` 为合同金额，`zsnd_yysr_bl` 为占上年营收比例（%）。
  - 多数字段为 `Option<String>`，缺失返回 null。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| start_date | string | Y | 起始日期（YYYY-MM-DD），区间跨度 ≤ 3 天 |
| end_date | string | Y | 结束日期（YYYY-MM-DD），区间跨度 ≤ 3 天 |

## 输出参数

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| code | int | Y | 状态码：0 成功 |
| message | string | Y | 状态信息 |
| data | object | Y | MajorContractData（见下） |

data ：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| total | int | Y | 合同总数 |
| items | array[MajorContractItem] | Y | 重大合同列表 |

MajorContractItem：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| seq | int | Y | 序号 |
| security_code | string | Y | 证券代码 |
| security_short_name | string | Y | 证券简称 |
| signatory | string | Y | 签约方 |
| signatory_rel_name | string | Y | 签约方关系 |
| counter_party | string | Y | 合同对方 |
| counter_party_rel_name | string | Y | 合同对方关系 |
| contract_type_name | string | Y | 合同类型（如「销售合同」） |
| contract_name | string | Y | 合同名称 |
| amounts | string | Y | 合同金额 |
| snd_yysr | string | Y | 上年同期营收 |
| zsnd_yysr_bl | string | Y | 占上年同期营收比例（%） |
| operate_reve | string | Y | 营业收入 |
| sign_date | string | Y | 签约日期 |
| dim_rdate | string | Y | 披露日期 |

## 调用方法（MCP）

> MCP 工具名 `ft_major_contract`。MCP Streamable HTTP 要求**先 initialize 拿 `Mcp-Session-Id`，再 `tools/call`**，后续请求带该 header。返回 **markdown 表格文本**。

**curl**：

```bash
SID=$(curl -sS -m 10 -D - -o /dev/null -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"t","version":"1"}}}' \
  | grep -i 'mcp-session-id:' | awk '{print $2}' | tr -d '')

curl -sS -m 10 -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -H "Mcp-Session-Id: $SID" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ft_major_contract","arguments":{"start_date": "2026-06-20", "end_date": "2026-06-23"}}}'
```

**Python（`mcp` SDK，自动握手管理 session）**：

```python
import asyncio
from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client

async def main():
    async with streamablehttp_client('<MCP_BASE_URL>') as (r, w, _):
        async with ClientSession(r, w) as s:
            await s.initialize()
            res = await s.call_tool('ft_major_contract', {"start_date": "2026-06-20", "end_date": "2026-06-23"})
            print(res.content[0].text)   # markdown 表格文本

asyncio.run(main())
```
