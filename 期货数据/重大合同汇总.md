# 重大合同汇总（MCP 工具 `ft_major_contract_summary`）

> **MCP 工具**：`ft_major_contract_summary`（category: `期货数据`）。返回 **markdown 表格文本**（非结构化 JSON）。输入参数 / 输出参数 / 数据样例对照 [ftshare-doc](../../ftshare-doc/api-doc) 原接口。

- 描述：近一年 A 股上市公司重大合同汇总，按证券代码汇总该年度的合同笔数、合同金额合计、上一年金额合计、去年营收、营收占比、最新营收等。数据来源于东方财富（eastmoney）重大合同汇总数据。响应为 `PaginatedApiResponse` 分页信封，data 含 `pageNum/pageSize/total/pages/records`。
- 数据范围：最新快照（近一年汇总）
- 单次限量：内存分页，默认 `page=1`、`page_size=50`、最大 200
- 提示：
  - 请求无业务必填参数，仅分页。
  - 汇总维度为「近一年」，非按签约日区间筛选。
  - 多数字段为 `Option<String>`，缺失返回 null。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| page | int | N | 页码，默认 1 |
| page_size | int | N | 每页条数，默认 50、最大 200 |

## 输出参数

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| code | int | Y | 状态码：0 成功 |
| message | string | Y | 状态信息 |
| data | object | Y | PaginatedData（见下） |

data ：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| pageNum | int | Y | 当前页码 |
| pageSize | int | Y | 每页条数 |
| total | int | Y | 总记录数 |
| pages | int | Y | 总页数 |
| records | array[MajorContractSummaryItem] | Y | 重大合同汇总列表 |

MajorContractSummaryItem：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| seq | int | Y | 序号 |
| security_code | string | Y | 证券代码 |
| security_short_name | string | Y | 证券简称 |
| contract_count | string | Y | 合同笔数 |
| total_amount | string | Y | 合同金额合计 |
| prev_year_total | string | Y | 上一年金额合计 |
| last_year_revenue | string | Y | 去年营收 |
| revenue_ratio | string | Y | 营收占比（%） |
| latest_revenue | string | Y | 最新营收 |

## 调用方法（MCP）

> MCP 工具名 `ft_major_contract_summary`。MCP Streamable HTTP 要求**先 initialize 拿 `Mcp-Session-Id`，再 `tools/call`**，后续请求带该 header。返回 **markdown 表格文本**。

**curl**：

```bash
SID=$(curl -sS -m 10 -D - -o /dev/null -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"t","version":"1"}}}' \
  | grep -i 'mcp-session-id:' | awk '{print $2}' | tr -d '')

curl -sS -m 10 -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -H "Mcp-Session-Id: $SID" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ft_major_contract_summary","arguments":{}}}'
```

**Python（`mcp` SDK，自动握手管理 session）**：

```python
import asyncio
from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client

async def main():
    async with streamablehttp_client('<MCP_BASE_URL>') as (r, w, _):
        async with ClientSession(r, w) as s:
            await s.initialize()
            res = await s.call_tool('ft_major_contract_summary', {})
            print(res.content[0].text)   # markdown 表格文本

asyncio.run(main())
```
