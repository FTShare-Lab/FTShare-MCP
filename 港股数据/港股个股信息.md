# 港股个股信息（MCP 工具 `ft_get_hk_basinfo`）

> **MCP 工具**：`ft_get_hk_basinfo`（category: `港股数据/基础数据`）。返回 **markdown 表格文本**（非结构化 JSON）。输入参数 / 输出参数 / 数据样例对照 [ftshare-doc](../../ftshare-doc/api-doc) 原接口。

- 描述：按 `hk_code` 查询港股个股信息视图（基础信息 + 最新市值/估值合并视图），返回单条 `HkStockViewResponse`。字段含板块/上市状态/关联 A 股代码、总股本/总市值/港股市值、PE/PS/PB/PCF/股息率等（数值字段为 `f64`）。GET 与 POST 走同一 inner 逻辑，返回结构一致。`hk_code` 必填，空值或查不到/底层日线文件缺失返回 400，其它内部错误返回 500。
- 数据范围：无时间维度，实时/最新快照（最新交易日视图）
- 单次限量：单条返回，无分页
- 提示：
  - 对外 v1 仅暴露 GET（v1 调 get_hk_stock_view；底层 v0 同时支持 GET+POST）。
  - `hk_code` 建议使用带 `.HK` 后缀的港股代码，如 `00700.HK`。
  - 数值字段为 `f64`（非 Decimal），与 F9 `market-cap-hk` 字段口径类似但类型不同。
  - 底层依赖日线静态文件，文件缺失返回 400。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| hk_code | string | Y | 港股代码，如 `00700.HK` |

## 调用方法（MCP）

> MCP 工具名 `ft_get_hk_basinfo`。MCP Streamable HTTP 要求**先 initialize 拿 `Mcp-Session-Id`，再 `tools/call`**，后续请求带该 header。返回 **markdown 表格文本**。

**curl**：

```bash
SID=$(curl -sS -m 10 -D - -o /dev/null -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"t","version":"1"}}}' \
  | grep -i 'mcp-session-id:' | awk '{print $2}' | tr -d '')

curl -sS -m 10 -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -H "Mcp-Session-Id: $SID" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ft_get_hk_basinfo","arguments":{"hk_code": "00700.HK"}}}'
```

**Python（`mcp` SDK，自动握手管理 session）**：

```python
import asyncio
from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client

async def main():
    async with streamablehttp_client('<MCP_BASE_URL>') as (r, w, _):
        async with ClientSession(r, w) as s:
            await s.initialize()
            res = await s.call_tool('ft_get_hk_basinfo', {"hk_code": "00700.HK"})
            print(res.content[0].text)   # markdown 表格文本

asyncio.run(main())
```
