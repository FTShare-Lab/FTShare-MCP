# 港股市值（MCP 工具 `ft_get_market_cap_hk`）

> **MCP 工具**：`ft_get_market_cap_hk`（category: `港股数据/行情数据`）。返回 **markdown 表格文本**（非结构化 JSON）。输入参数 / 输出参数 / 数据样例对照 [ftshare-doc](../../ftshare-doc/api-doc) 原接口。

- 描述：按 `trade_code` 查询港股市值/估值分析最新一期数据，返回单条 `MarketCapHkResponse`。含总股本（= H 股 + A 股 + B 股股数）、总市值（= 总股本 × 收盘价）、港股市值/流通市值、静态/动态/滚动 PE、PS、PB、PCF、股息率等估值指标。无分页，`trade_code` 必填，查不到报 `trade_code not found`。
- 数据范围：无时间维度，实时/最新快照（最新一期估值）
- 单次限量：单条返回，无分页
- 提示：
  - 对外 v1 仅暴露 GET（底层 v0 同时支持 GET+POST）。
  - `trade_code` 必填，建议使用带 `.HK` 后缀的港股代码，如 `00700.HK`。
  - 仅返回最新一期数据，非时间序列；如需历史估值见 F12 `hk-valuatnanalyd`。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| trade_code | string | Y | 港股交易代码 |

## 输出参数

单条对象（MarketCapHkResponse）：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| trade_code | string | Y | 交易代码 |
| security_name | string | Y | 证券简称 |
| trade_date | int32 | Y | 计算日期 YYYYMMDD |
| total_share_num | decimal | Y | 总股本 = hshare_num + ashare_num + bshare_num |
| total_mv | decimal | Y | 总市值 = 总股本 × price_close |
| hshare_num | decimal | Y | 港股股数 |
| hshare_limit_num | decimal | Y | 港股限售股股数 |
| hshare_mv | decimal | Y | 港股市值（元） |
| hshare_circ_mv | decimal | Y | 港股流通市值（元） |
| nonhshare_num | decimal | Y | 非港股股数 |
| ashare_num | decimal | Y | A 股股数 |
| bshare_num | decimal | Y | B 股股数 |
| pe_lyr | decimal | Y | 静态市盈率 |
| forwd_pe | decimal | Y | 动态市盈率 |
| pe_ttm | decimal | Y | 滚动市盈率 |
| ps | decimal | Y | 市销率 |
| ps_ttm | decimal | Y | 滚动市销率 |
| pb | decimal | Y | 市净率 |
| pcf | decimal | Y | 市现率（经营现金流量净额） |
| pcf_ttm | decimal | Y | 滚动市现率（经营现金流量净额） |
| dividrt_lyr_rpt | decimal | Y | 静态股息率（报告期，%） |
| dividrt_ttm | decimal | Y | 滚动股息率（%） |

## 调用方法（MCP）

> MCP 工具名 `ft_get_market_cap_hk`。MCP Streamable HTTP 要求**先 initialize 拿 `Mcp-Session-Id`，再 `tools/call`**，后续请求带该 header。返回 **markdown 表格文本**。

**curl**：

```bash
SID=$(curl -sS -m 10 -D - -o /dev/null -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"t","version":"1"}}}' \
  | grep -i 'mcp-session-id:' | awk '{print $2}' | tr -d '')

curl -sS -m 10 -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -H "Mcp-Session-Id: $SID" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ft_get_market_cap_hk","arguments":{"trade_code": "00700.HK"}}}'
```

**Python（`mcp` SDK，自动握手管理 session）**：

```python
import asyncio
from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client

async def main():
    async with streamablehttp_client('<MCP_BASE_URL>') as (r, w, _):
        async with ClientSession(r, w) as s:
            await s.initialize()
            res = await s.call_tool('ft_get_market_cap_hk', {"trade_code": "00700.HK"})
            print(res.content[0].text)   # markdown 表格文本

asyncio.run(main())
```
