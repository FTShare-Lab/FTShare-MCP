# 贵金属价格（MCP 工具 `ft_get_bullion_price`）

> **MCP 工具**：`ft_get_bullion_price`（category: `现货数据`）。返回 **markdown 表格文本**（非结构化 JSON）。输入参数 / 输出参数 / 数据样例对照 [ftshare-doc](../../ftshare-doc/api-doc) 原接口。

- 描述：查询指定标的的贵金属行情（日期 + 值），按 `start_date`/`end_date` 区间分页返回 `PaginatedResponse<BullionPriceItem>`。handler 校验 `start_date`/`end_date` 区间跨度 ≤3 天（`validate_yyyymmdd_range_3_days`），由 `query_bullion_price` 查询 ClickHouse。
- 数据范围：以服务端返回为准
- 单次限量：分页返回，由 `page`/`page_size` 控制；`start_date`/`end_date` 跨度 ≤3 天
- 提示：
  - `symbol`、`start_date`、`end_date`、`page`、`page_size` 均需传入。
  - `start_date`/`end_date` 为 YYYYMMDD，区间跨度硬限制 ≤3 天。
  - `value` 以字符串形式序列化（高精度 Decimal）。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| symbol | string | Y | 标的代码，如 XAUUSD、AU9999 |
| start_date | int | Y | 查询起始日期 YYYYMMDD |
| end_date | int | Y | 查询结束日期 YYYYMMDD；与 start_date 跨度 ≤3 天 |
| page | int | Y | 当前页 |
| page_size | int | Y | 每页条数 |

## 输出参数

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| items | array[BullionPriceItem] | Y | 行情列表 |
| total_pages | int | Y | 总页数 |
| total_items | int | Y | 总记录数（未分页前） |

BullionPriceItem：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| date | string | Y | 日期 YYYY-MM-DD |
| value | string(decimal) | Y | 值（高精度 Decimal 序列化为字符串） |

## 调用方法（MCP）

> MCP 工具名 `ft_get_bullion_price`。MCP Streamable HTTP 要求**先 initialize 拿 `Mcp-Session-Id`，再 `tools/call`**，后续请求带该 header。返回 **markdown 表格文本**。

**curl**：

```bash
SID=$(curl -sS -m 10 -D - -o /dev/null -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"t","version":"1"}}}' \
  | grep -i 'mcp-session-id:' | awk '{print $2}' | tr -d '')

curl -sS -m 10 -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -H "Mcp-Session-Id: $SID" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ft_get_bullion_price","arguments":{"symbol": "XAUUSD", "start_date": 20260620, "end_date": 20260623, "page": 1, "page_size": 3}}}'
```

**Python（`mcp` SDK，自动握手管理 session）**：

```python
import asyncio
from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client

async def main():
    async with streamablehttp_client('<MCP_BASE_URL>') as (r, w, _):
        async with ClientSession(r, w) as s:
            await s.initialize()
            res = await s.call_tool('ft_get_bullion_price', {"symbol": "XAUUSD", "start_date": 20260620, "end_date": 20260623, "page": 1, "page_size": 3})
            print(res.content[0].text)   # markdown 表格文本

asyncio.run(main())
```
