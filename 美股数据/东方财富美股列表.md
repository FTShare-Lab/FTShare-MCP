# 东方财富美股列表（MCP 工具 `ft_eastmoney_us_stock_list`）

> **MCP 工具**：`ft_eastmoney_us_stock_list`（category: `美股数据/行情数据`）。返回 **markdown 表格文本**（非结构化 JSON）。输入参数 / 输出参数 / 数据样例对照 [ftshare-doc](../../ftshare-doc/api-doc) 原接口。

- 描述：获取东方财富**美股列表**（从内存缓存返回），含证券 ID（secid `{market}.{code}`）、市值（美元）、最新价、涨跌幅、成交量、成交额、市盈率 TTM 等。数据来源：东方财富，本地维护 CSV 并加载到内存缓存。支持分页；`refresh=true` 时强制从 CSV 重新加载缓存。响应为 `PaginatedResponse`（非信封，顶层即分页对象）。
- 数据范围：列表类，最新快照（无时间维度）
- 单次限量：分页返回，默认 `page=1`、`page_size` 由后端定
- 提示：
  - 列表数据由本地 CSV 缓存驱动，调用方传 `refresh=true` 会触发一次磁盘重载（较慢，慎用）。
  - secid 格式 `{market}.{code}`，如 `105.ADV`。
  - 响应结构与东方财富资金流/估值接口不同——本接口直接是 `PaginatedResponse`（items/total_pages/total_items），无 code/message 信封。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| refresh | bool | N | 为 true 时强制从 CSV 重新加载列表缓存 |
| page | int | N | 页码，从 1 开始 |
| page_size | int | N | 每页数量 |

## 输出参数

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| items | array[EastmoneyUsStock] | Y | 美股列表 |
| total_pages | int | Y | 总页数 |
| total_items | int | Y | 总记录数（未分页前） |

EastmoneyUsStock：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| secid | string | Y | 证券 ID，格式 {market}.{code}，如 105.ADV |
| market | string | Y | 市场编号，如 105 |
| code | string | Y | 股票代码，如 ADV |
| name | string | Y | 股票名称 |
| market_value_usd | string | Y | 市值（美元） |
| latest_price | string | Y | 最新价 |
| change_pct | string | Y | 涨跌幅 |
| volume | string | Y | 成交量 |
| amount | string | Y | 成交额 |
| pe_ttm | string | Y | 市盈率 TTM |

## 调用方法（MCP）

> MCP 工具名 `ft_eastmoney_us_stock_list`。MCP Streamable HTTP 要求**先 initialize 拿 `Mcp-Session-Id`，再 `tools/call`**，后续请求带该 header。返回 **markdown 表格文本**。

**curl**：

```bash
SID=$(curl -sS -m 10 -D - -o /dev/null -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"t","version":"1"}}}' \
  | grep -i 'mcp-session-id:' | awk '{print $2}' | tr -d '')

curl -sS -m 10 -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -H "Mcp-Session-Id: $SID" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ft_eastmoney_us_stock_list","arguments":{}}}'
```

**Python（`mcp` SDK，自动握手管理 session）**：

```python
import asyncio
from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client

async def main():
    async with streamablehttp_client('<MCP_BASE_URL>') as (r, w, _):
        async with ClientSession(r, w) as s:
            await s.initialize()
            res = await s.call_tool('ft_eastmoney_us_stock_list', {})
            print(res.content[0].text)   # markdown 表格文本

asyncio.run(main())
```
