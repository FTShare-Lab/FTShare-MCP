# 业绩快报（MCP 工具 `ft_earnings_reports_paginated`）

> **MCP 工具**：`ft_earnings_reports_paginated`（category: `股票数据/财务数据`）。返回 **markdown 表格文本**（非结构化 JSON）。输入参数 / 输出参数 / 数据样例对照 [ftshare-doc](../../ftshare-doc/api-doc) 原接口。

- 描述：查询 A 股业绩快报（`ashare_perf_quick`），返回已披露的业绩快报核心指标（EPS、营业总收入、净利润、ROE 等，含本期/上期/同比）。数据走 Redis 缓存。支持两种查询方式：① 传 `stock_code` 查单票全部业绩快报（不分页）；② 传 `year + report_type` 按报告期分页查所有票。`report_type` 为枚举 `ReportType`：`q1`（一季报）/ `q2`（半年报）/ `q3`（三季报）/ `annual`（年报）/ `announcement`。
- 数据范围：以服务端返回为准
- 单次限量：默认 `page=1`、`page_size=50`，最大 500（`validate_page` 校验）
- 提示：
  - `stock_code` 与 `year + report_type` 二选一；都不传返回空结果。
  - 单票查询（`stock_code`）不走分页，`total_pages` 固定 1。
  - `report_type` 枚举值为小写：`q1` / `q2` / `q3` / `annual` / `announcement`。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| stock_code | string | N | A 股代码（如 000001.SZ）；传则查单票全部业绩快报 |
| year | int | N | 年份（按报告期查询时必填），如 2024 |
| report_type | string | N | 报告类型枚举（按报告期查询时必填）：q1 / q2 / q3 / annual / announcement |
| page | int | N | 页码，默认 1 |
| page_size | int | N | 每页条数，默认 50，最大 500 |

## 输出参数

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| items | array | Y | 当前页记录列表 |
| total_pages | int | Y | 总页数 |
| total_items | int | Y | 总记录数 |

## 调用方法（MCP）

> MCP 工具名 `ft_earnings_reports_paginated`。MCP Streamable HTTP 要求**先 initialize 拿 `Mcp-Session-Id`，再 `tools/call`**，后续请求带该 header。返回 **markdown 表格文本**。

**curl**：

```bash
SID=$(curl -sS -m 10 -D - -o /dev/null -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"t","version":"1"}}}' \
  | grep -i 'mcp-session-id:' | awk '{print $2}' | tr -d '')

curl -sS -m 10 -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -H "Mcp-Session-Id: $SID" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ft_earnings_reports_paginated","arguments":{}}}'
```

**Python（`mcp` SDK，自动握手管理 session）**：

```python
import asyncio
from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client

async def main():
    async with streamablehttp_client('<MCP_BASE_URL>') as (r, w, _):
        async with ClientSession(r, w) as s:
            await s.initialize()
            res = await s.call_tool('ft_earnings_reports_paginated', {})
            print(res.content[0].text)   # markdown 表格文本

asyncio.run(main())
```
