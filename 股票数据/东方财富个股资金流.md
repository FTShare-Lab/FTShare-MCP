# 东方财富个股资金流（MCP 工具 `ft_get_eastmoney_stock_flow`）

> **MCP 工具**：`ft_get_eastmoney_stock_flow`（category: `股票数据/资金流向数据`）。返回 **markdown 表格文本**（非结构化 JSON）。输入参数 / 输出参数 / 数据样例对照 [ftshare-doc](../../ftshare-doc/api-doc) 原接口。

- 描述：获取东方财富**个股日资金流**数据，含主力/超大单/大单/中单/小单四档净流入及净占比，以及收盘价与涨跌幅。数据来源：东方财富，底表为 ClickHouse `basedata.stock_daily_flow`。支持按股票代码、交易日或日期区间过滤。响应为 `PaginatedApiResponse` 信封（code/message/data）。
- 数据范围：资金流类，最新一期或当日快照（历史序列下限以服务端返回为准）
- 单次限量：分页返回，默认 `page=1`、`page_size=50`、最大 `page_size=200`
- 提示：
  - `symbol` 为纯数字股票代码（如 600522），不带交易所后缀。
  - `market` 字段：0=深/京，1=沪。
  - `trade_date` 与 `start_date`/`end_date` 互斥。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| symbol | string | N | 股票代码，如 600522 |
| trade_date | string | N | 交易日 YYYYMMDD；与 start_date/end_date 互斥 |
| start_date | string | N | 区间起始日 YYYYMMDD；需与 end_date 同时提供 |
| end_date | string | N | 区间结束日 YYYYMMDD；需与 start_date 同时提供 |
| page | int | N | 页码，从 1 开始，默认 1 |
| page_size | int | N | 每页条数，默认 50，最大 200 |

## 输出参数

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| code | int | Y | 状态码：0 成功 |
| message | string | Y | 状态信息 |
| data | object | Y | 分页对象（见下） |

data ：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| pageNum | int | Y | 当前页码 |
| pageSize | int | Y | 每页条数 |
| total | int | Y | 总记录数 |
| pages | int | Y | 总页数 |
| records | array[EastmoneyStockFlowItem] | Y | 个股资金流列表 |

EastmoneyStockFlowItem：

| 名称 | 类型 | 默认显示 | 描述 |

| trade_date | string | Y | 交易日期 YYYYMMDD |

| change_pct | string | Y | 涨跌幅（%） |

## 调用方法（MCP）

> MCP 工具名 `ft_get_eastmoney_stock_flow`。MCP Streamable HTTP 要求**先 initialize 拿 `Mcp-Session-Id`，再 `tools/call`**，后续请求带该 header。返回 **markdown 表格文本**。

**curl**：

```bash
SID=$(curl -sS -m 10 -D - -o /dev/null -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"t","version":"1"}}}' \
  | grep -i 'mcp-session-id:' | awk '{print $2}' | tr -d '')

curl -sS -m 10 -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -H "Mcp-Session-Id: $SID" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ft_get_eastmoney_stock_flow","arguments":{}}}'
```

**Python（`mcp` SDK，自动握手管理 session）**：

```python
import asyncio
from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client

async def main():
    async with streamablehttp_client('<MCP_BASE_URL>') as (r, w, _):
        async with ClientSession(r, w) as s:
            await s.initialize()
            res = await s.call_tool('ft_get_eastmoney_stock_flow', {})
            print(res.content[0].text)   # markdown 表格文本

asyncio.run(main())
```
