# 东方财富股东增减持（MCP 工具 `ft_stock_ggcg_em_handler`）

> **MCP 工具**：`ft_stock_ggcg_em_handler`（category: `股票数据/参考数据`）。返回 **markdown 表格文本**（非结构化 JSON）。输入参数 / 输出参数 / 数据样例对照 [ftshare-doc](../../ftshare-doc/api-doc) 原接口。

- 描述：东方财富股东增减持数据。接口名与 `symbol` 参数对齐 AkShare `stock_ggcg_em`，`symbol` 为数据类型筛选：`全部`/`股东增持`/`股东减持`，默认 `全部`。返回股东名称、变动数量/比例、变动后持股、变动起止与公告日期等。响应为 `PaginatedApiResponse` 信封（code/message/data），data 为分页对象（records/pageNum/pageSize/total/pages）。字段序列化为 camelCase。
- 数据范围：以服务端返回为准
- 单次限量：分页返回，默认 `page=1`、`page_size` 由后端定
- 提示：
  - `symbol` 默认 `全部`，可传 `股东增持` / `股东减持` 过滤。
  - 数据来源东方财富（来源标识 em）。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| symbol | string | N | 数据类型：全部 / 股东增持 / 股东减持，默认全部 |
| page | int | N | 页码 |
| page_size | int | N | 每页条数 |

## 输出参数

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| code | int | Y | 状态码：0 成功 |
| message | string | Y | 状态信息 |
| data | object | Y | 分页对象（见下） |

data ：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| pageNum | int | Y | 当前页码 |
| pageSize | int | Y | 每页条数 |
| total | int | Y | 总记录数 |
| pages | int | Y | 总页数 |
| records | array[StockGgcgEmItem] | Y | 增减持明细列表 |

StockGgcgEmItem：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| stock_code | string | Y | 股票代码 |
| stock_name | string | N | 股票名称 |
| latest_price | string | N | 最新价 |
| price_change_rate | string | N | 涨跌幅 |
| holder_name | string | Y | 股东名称 |
| change_num_symbol | string | N | 变动数量（带符号） |
| change_num | string | N | 变动数量 |
| change_total_ratio | string | N | 变动占总股本比例 |
| change_free_ratio | string | N | 变动占流通股本比例 |

| hold_ratio | string | N | 持股比例 |
| free_shares | string | N | 流通股 |

| start_date | string | N | 变动起始日期 |
| end_date | string | N | 变动截止日期 |
| notice_date | string | N | 公告日期 |

## 调用方法（MCP）

> MCP 工具名 `ft_stock_ggcg_em_handler`。MCP Streamable HTTP 要求**先 initialize 拿 `Mcp-Session-Id`，再 `tools/call`**，后续请求带该 header。返回 **markdown 表格文本**。

**curl**：

```bash
SID=$(curl -sS -m 10 -D - -o /dev/null -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"t","version":"1"}}}' \
  | grep -i 'mcp-session-id:' | awk '{print $2}' | tr -d '')

curl -sS -m 10 -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -H "Mcp-Session-Id: $SID" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ft_stock_ggcg_em_handler","arguments":{}}}'
```

**Python（`mcp` SDK，自动握手管理 session）**：

```python
import asyncio
from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client

async def main():
    async with streamablehttp_client('<MCP_BASE_URL>') as (r, w, _):
        async with ClientSession(r, w) as s:
            await s.initialize()
            res = await s.call_tool('ft_stock_ggcg_em_handler', {})
            print(res.content[0].text)   # markdown 表格文本

asyncio.run(main())
```
