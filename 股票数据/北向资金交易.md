# 北向资金交易（MCP 工具 `ft_northbound`）

> **MCP 工具**：`ft_northbound`（category: `股票数据/资金流向数据`）。返回 **markdown 表格文本**（非结构化 JSON）。输入参数 / 输出参数 / 数据样例对照 [ftshare-doc](../../ftshare-doc/api-doc) 原接口。

- 描述：查询北向资金（沪深股通）交易日交易数据。按交易日返回当日北向资金成交总额、币种，以及沪股通（SH）/ 深股通（SZ）分市场通道的成交额与成交笔数。`date` 必填（YYYYMMDD），底层从南北向数据模块按日期加载。响应为 `NorthboundResponseResponse` 信封（code/message/data），data 中含日期、币种、总成交额与分市场通道。
- 数据范围：按交易日，以服务端返回为准
- 单次限量：单个交易日一条聚合数据
- 提示：
  - `date` 必填，格式 YYYYMMDD。
  - `amount` / `total_amount` 单位元（币种见 `currency` 字段）。
  - 响应为 code/message/data 信封结构。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| date | string | Y | 交易日，格式 YYYYMMDD |

## 输出参数

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| code | int | Y | 状态码：0 成功 |
| message | string | Y | 状态信息 |
| data | object | Y | 北向资金交易数据（见下） |

data ：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| date | string | Y | 交易日 |
| currency | string | Y | 币种 |
| total_amount | string | Y | 北向资金总成交额，单位元 |
| channels | object | Y | 分市场通道（见下） |

channels ：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| SH | object | Y | 沪股通通道数据 |
| SZ | object | Y | 深股通通道数据 |

NorthboundChannelDataResponse：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| amount | string | Y | 该通道成交额，单位元 |
| trade_count | int | Y | 该通道成交笔数 |

## 调用方法（MCP）

> MCP 工具名 `ft_northbound`。MCP Streamable HTTP 要求**先 initialize 拿 `Mcp-Session-Id`，再 `tools/call`**，后续请求带该 header。返回 **markdown 表格文本**。

**curl**：

```bash
SID=$(curl -sS -m 10 -D - -o /dev/null -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"t","version":"1"}}}' \
  | grep -i 'mcp-session-id:' | awk '{print $2}' | tr -d '')

curl -sS -m 10 -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -H "Mcp-Session-Id: $SID" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ft_northbound","arguments":{"date": "20260624"}}}'
```

**Python（`mcp` SDK，自动握手管理 session）**：

```python
import asyncio
from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client

async def main():
    async with streamablehttp_client('<MCP_BASE_URL>') as (r, w, _):
        async with ClientSession(r, w) as s:
            await s.initialize()
            res = await s.call_tool('ft_northbound', {"date": "20260624"})
            print(res.content[0].text)   # markdown 表格文本

asyncio.run(main())
```
