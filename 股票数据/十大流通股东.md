# 十大流通股东（MCP 工具 `ft_stock_float_holders`）

> **MCP 工具**：`ft_stock_float_holders`（category: `股票数据/参考数据`）。返回 **markdown 表格文本**（非结构化 JSON）。输入参数 / 输出参数 / 数据样例对照 [ftshare-doc](../../ftshare-doc/api-doc) 原接口。

- 描述：查询 A 股十大流通股东。支持两种模式：① 指定 `stock_code` 返回该标的的全部历史十大流通股东记录；② `is_last=true` 时返回所有标的最新一期的十大流通股东数据（分页）。数据从 Redis 缓存读取（底层数据源 ClickHouse `asare_top10_float_holder` 表）。响应为 `PaginatedResponse` 分页结构（items/total_pages/total_items）。
- 数据范围：按报告期，以服务端返回为准
- 单次限量：默认 `page=1`、`page_size=50`；按 `stock_code` 查询时不分页，一次返回该标的全量历史
- 提示：
  - 未指定 `stock_code` 时，需配合 `is_last=true` 才有数据返回。
  - `fen_holders` 为该报告期前十大流通股东明细数组。
  - 数据更新随上市公司定期报告披露。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| stock_code | string | N | 标的代码，指定时返回该标的全部历史数据 |
| is_last | bool | N | 是否取所有标的最新一期，true 时按 page/page_size 分页 |
| page | int | N | 页码，默认 1 |
| page_size | int | N | 每页条数，默认 50 |

## 输出参数

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| items | array[TopTenHolders] | Y | 十大流通股东列表 |
| total_pages | int | Y | 总页数 |
| total_items | int | Y | 总记录数 |

TopTenHolders：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| stock_code | string | Y | 股票代码 |
| stock_name | string | Y | 股票全称 |
| publish_date | string | Y | 发布日期 YYYY-MM-DD |
| share_holding | decimal | Y | 持股合计 |
| fen_holders | array[HolderDetail] | Y | 十大流通股东明细 |

HolderDetail：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| rank | int | Y | 股东名次 |
| shareholder_name | string | Y | 股东名称 |
| shareholder_type | string | Y | 股东性质 |
| share_type | string | Y | 股份类型 |
| shareholding | decimal | Y | 持股数（股） |
| share_ratio | decimal | Y | 占股本持股比例（%） |
| limit_num | decimal | N | 限售数量 |
| unlimit_num | decimal | N | 无限售数量 |
| change_shares | decimal | Y | 增减（股） |
| change_type | string | N | 变动类型 |
| change_percentage | decimal | N | 变动比例（%） |

## 调用方法（MCP）

> MCP 工具名 `ft_stock_float_holders`。MCP Streamable HTTP 要求**先 initialize 拿 `Mcp-Session-Id`，再 `tools/call`**，后续请求带该 header。返回 **markdown 表格文本**。

**curl**：

```bash
SID=$(curl -sS -m 10 -D - -o /dev/null -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"t","version":"1"}}}' \
  | grep -i 'mcp-session-id:' | awk '{print $2}' | tr -d '')

curl -sS -m 10 -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -H "Mcp-Session-Id: $SID" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ft_stock_float_holders","arguments":{}}}'
```

**Python（`mcp` SDK，自动握手管理 session）**：

```python
import asyncio
from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client

async def main():
    async with streamablehttp_client('<MCP_BASE_URL>') as (r, w, _):
        async with ClientSession(r, w) as s:
            await s.initialize()
            res = await s.call_tool('ft_stock_float_holders', {})
            print(res.content[0].text)   # markdown 表格文本

asyncio.run(main())
```
