# 单标的集合竞价成交量（MCP 工具 `ft_stock_intraday_auction_volume_symbol`）

> **MCP 工具**：`ft_stock_intraday_auction_volume_symbol`（category: `股票数据/特色数据`）。返回 **markdown 表格文本**（非结构化 JSON）。输入参数 / 输出参数 / 数据样例对照 [ftshare-doc](../../ftshare-doc/api-doc) 原接口。

- 描述：查询单只标的的连续竞价成交占比（个股模式），按分钟返回该标的当日（或指定交易日）成交量/成交额占比分时序列。不传 `trade_date` 时返回当日实时个股数据；传入 `trade_date=YYYYMMDD` 时返回历史分钟线派生结果；若 `trade_date` 为今日则自动降级为实时数据（历史表尚未落库）。必须同时指定 `symbol` 与可选 `trade_date`；缺 `symbol` 时返回空列表。`symbol` 格式为 `code.suffix`（suffix 支持 XSHG/SH、XSHE/SZ、BJSE/BJ）。响应为 `PaginatedApiResponse<AuctionVolumeItem>`（code/message/data 信封 + 分页），每条为一分钟的个股快照。
- 数据范围：以服务端返回为准（按交易日分钟级更新）
- 单次限量：分页返回，默认 `page=1`、`page_size=50`，`page_size` 最大 200
- 提示：
  - `symbol` 格式 `code.suffix`（如 `600000.SH`），suffix 支持 XSHG/SH、XSHE/SZ、BJSE/BJ。
  - `trade_date` 格式 YYYYMMDD；不传返回当日实时数据，传入当日会降级为实时。
  - `volume_ratio` = 该分钟成交量 / 全天总成交量；`turnover_ratio` = 该分钟成交额 / 全天总成交额。
  - 缺 `symbol` 返回空 records。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| symbol | string | Y | 标的代码，格式 code.suffix（XSHG/SH、XSHE/SZ、BJSE/BJ） |
| trade_date | string | N | 交易日期 YYYYMMDD；不传返回当日实时数据 |
| page | int | N | 页码，从 1 开始，默认 1 |
| page_size | int | N | 每页数量，默认 50，最大 200 |

## 输出参数

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| code | int | Y | 状态码：0 成功 |
| message | string | Y | 状态信息 |
| data | object | N | 分页对象（见下） |

data ：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| page_num | int | Y | 当前页码 |
| page_size | int | Y | 每页条数 |
| total | int | Y | 总记录数 |
| pages | int | Y | 总页数 |
| records | array[AuctionVolumeItem] | Y | 分钟级个股列表 |

AuctionVolumeItem：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| ts_millis | int | Y | 该分钟收盘时间戳，单位毫秒 |
| volume | int | Y | 该分钟成交量 |
| volume_ratio | f64 | Y | 成交量占比 = 该分钟成交量 / 全天总成交量 |
| turnover | string | Y | 该分钟成交额，单位元 |
| turnover_ratio | f64 | Y | 成交额占比 = 该分钟成交额 / 全天总成交额 |

## 调用方法（MCP）

> MCP 工具名 `ft_stock_intraday_auction_volume_symbol`。MCP Streamable HTTP 要求**先 initialize 拿 `Mcp-Session-Id`，再 `tools/call`**，后续请求带该 header。返回 **markdown 表格文本**。

**curl**：

```bash
SID=$(curl -sS -m 10 -D - -o /dev/null -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"t","version":"1"}}}' \
  | grep -i 'mcp-session-id:' | awk '{print $2}' | tr -d '')

curl -sS -m 10 -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -H "Mcp-Session-Id: $SID" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ft_stock_intraday_auction_volume_symbol","arguments":{"symbol": "600584.SH"}}}'
```

**Python（`mcp` SDK，自动握手管理 session）**：

```python
import asyncio
from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client

async def main():
    async with streamablehttp_client('<MCP_BASE_URL>') as (r, w, _):
        async with ClientSession(r, w) as s:
            await s.initialize()
            res = await s.call_tool('ft_stock_intraday_auction_volume_symbol', {"symbol": "600584.SH"})
            print(res.content[0].text)   # markdown 表格文本

asyncio.run(main())
```
