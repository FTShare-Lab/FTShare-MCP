# 商誉减值（MCP 工具 `ft_goodwill_stock_impairment`）

> **MCP 工具**：`ft_goodwill_stock_impairment`（category: `股票数据/财务数据`）。返回 **markdown 表格文本**（非结构化 JSON）。输入参数 / 输出参数 / 数据样例对照 [ftshare-doc](../../ftshare-doc/api-doc) 原接口。

- 描述：按报告期查询 A 股个股商誉减值明细，返回每只股票的商誉规模、商誉变动、商誉占净资产比例、净利润规模、商誉减值占净利润比例、公告日期、所属板块。数据来源于商誉减值表。响应为 `PaginatedApiResponse` 分页信封，data 含 `pageNum/pageSize/total/pages/records`。
- 数据范围：按报告期查询（`date` 必填，如 `20250331`）
- 单次限量：内存分页，默认 `page=1`、`page_size=50`、最大 200
- 提示：
  - `date` 必填，报告期通常为季末/年末。
  - 个股维度商誉减值明细。
  - `goodwill_change` 为负表示减值；多数字段为 `Option<String>`，缺失返回 null。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| date | string | Y | 报告期，如 20250331 |
| page | int | N | 页码，默认 1 |
| page_size | int | N | 每页条数，默认 50、最大 200 |

## 输出参数

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| code | int | Y | 状态码：0 成功 |
| message | string | Y | 状态信息 |
| data | object | Y | PaginatedData（见下） |

data ：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| pageNum | int | Y | 当前页码 |
| pageSize | int | Y | 每页条数 |
| total | int | Y | 总记录数 |
| pages | int | Y | 总页数 |
| records | array[GoodwillStockImpairmentItem] | Y | 个股商誉减值明细列表 |

GoodwillStockImpairmentItem：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| seq | int | Y | 序号 |
| security_code | string | Y | 证券代码 |
| security_name | string | Y | 证券简称 |
| goodwill_scale | string | Y | 商誉规模 |
| goodwill_change | string | Y | 商誉变动（负值表示减值） |
| goodwill_to_net_assets_ratio | string | Y | 商誉占净资产比例 |
| net_profit_scale | string | Y | 净利润规模 |
| goodwill_impairment_to_net_profit | string | Y | 商誉减值占净利润比例 |
| notice_date | string | Y | 公告日期 |
| trade_board | string | Y | 所属板块（如「主板」） |

## 调用方法（MCP）

> MCP 工具名 `ft_goodwill_stock_impairment`。MCP Streamable HTTP 要求**先 initialize 拿 `Mcp-Session-Id`，再 `tools/call`**，后续请求带该 header。返回 **markdown 表格文本**。

**curl**：

```bash
SID=$(curl -sS -m 10 -D - -o /dev/null -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"t","version":"1"}}}' \
  | grep -i 'mcp-session-id:' | awk '{print $2}' | tr -d '')

curl -sS -m 10 -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -H "Mcp-Session-Id: $SID" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ft_goodwill_stock_impairment","arguments":{"date": "20260624"}}}'
```

**Python（`mcp` SDK，自动握手管理 session）**：

```python
import asyncio
from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client

async def main():
    async with streamablehttp_client('<MCP_BASE_URL>') as (r, w, _):
        async with ClientSession(r, w) as s:
            await s.initialize()
            res = await s.call_tool('ft_goodwill_stock_impairment', {"date": "20260624"})
            print(res.content[0].text)   # markdown 表格文本

asyncio.run(main())
```
