# 商誉市场总览（MCP 工具 `ft_goodwill_market_overview`）

> **MCP 工具**：`ft_goodwill_market_overview`（category: `股票数据/财务数据`）。返回 **markdown 表格文本**（非结构化 JSON）。输入参数 / 输出参数 / 数据样例对照 [ftshare-doc](../../ftshare-doc/api-doc) 原接口。

- 描述：查询 A 股商誉市场总览，返回每个报告期的全市场商誉规模、商誉减值、净资产、商誉占净资产比例、减值占净资产比例、净利润规模、减值占净利润比例等总量指标。数据来源于商誉市场总览表。请求无业务参数。响应为 `ApiResponse` 信封（code/message/data），data 含 `total` 与 `items`。
- 数据范围：最新快照（按报告期序列）
- 单次限量：以服务端返回为准
- 提示：
  - 请求参数为空（`GoodwillMarketOverviewRequest {}`）。
  - 全市场总量维度，非个股/非行业。
  - 多数字段为 `Option<String>`，缺失返回 null。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| （无业务参数） | - | - | 请求体为空 |

## 输出参数

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| code | int | Y | 状态码：0 成功 |
| message | string | Y | 状态信息 |
| data | object | Y | GoodwillMarketOverviewData（见下） |

data ：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| total | int | Y | 报告期数 |
| items | array[GoodwillMarketOverviewItem] | Y | 各报告期市场总览列表 |

GoodwillMarketOverviewItem：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| report_date | string | Y | 报告期（如 20250331） |
| goodwill_scale | string | Y | 商誉规模 |
| goodwill_impairment | string | Y | 商誉减值 |
| net_assets | string | Y | 净资产规模 |
| goodwill_to_net_assets_ratio | string | Y | 商誉占净资产比例 |
| impairment_to_net_assets_ratio | string | Y | 减值占净资产比例 |
| net_profit_scale | string | Y | 净利润规模 |
| impairment_to_net_profit_ratio | string | Y | 减值占净利润比例 |

## 调用方法（MCP）

> MCP 工具名 `ft_goodwill_market_overview`。MCP Streamable HTTP 要求**先 initialize 拿 `Mcp-Session-Id`，再 `tools/call`**，后续请求带该 header。返回 **markdown 表格文本**。

**curl**：

```bash
SID=$(curl -sS -m 10 -D - -o /dev/null -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"t","version":"1"}}}' \
  | grep -i 'mcp-session-id:' | awk '{print $2}' | tr -d '')

curl -sS -m 10 -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -H "Mcp-Session-Id: $SID" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ft_goodwill_market_overview","arguments":{}}}'
```

**Python（`mcp` SDK，自动握手管理 session）**：

```python
import asyncio
from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client

async def main():
    async with streamablehttp_client('<MCP_BASE_URL>') as (r, w, _):
        async with ClientSession(r, w) as s:
            await s.initialize()
            res = await s.call_tool('ft_goodwill_market_overview', {})
            print(res.content[0].text)   # markdown 表格文本

asyncio.run(main())
```
