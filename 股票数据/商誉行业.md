# 商誉行业（MCP 工具 `ft_goodwill_industry`）

> **MCP 工具**：`ft_goodwill_industry`（category: `股票数据/财务数据`）。返回 **markdown 表格文本**（非结构化 JSON）。输入参数 / 输出参数 / 数据样例对照 [ftshare-doc](../../ftshare-doc/api-doc) 原接口。

- 描述：按报告期查询 A 股各行业的商誉汇总，返回每个行业的公司数、商誉规模、净资产、商誉占净资产比例、净利润规模。数据来源于商誉表。响应为 `ApiResponse` 信封（code/message/data），data 含查询 `date`、`total` 与 `items`。
- 数据范围：按报告期查询（`date` 必填，如 `20250331`）
- 单次限量：以服务端返回为准
- 提示：
  - `date` 必填，报告期通常为季末/年末（如 20250331、20241231）。
  - 行业维度汇总，非个股。
  - 多数字段为 `Option<T>`，缺失返回 null。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| date | string | Y | 报告期，如 20250331 |
| page | int | N | 页码（汇总数据，参考用） |
| page_size | int | N | 每页条数（汇总数据，参考用） |

## 输出参数

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| code | int | Y | 状态码：0 成功 |
| message | string | Y | 状态信息 |
| data | object | Y | GoodwillIndustryData（见下） |

data ：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| date | string | Y | 查询报告期（回显） |
| total | int | Y | 行业数 |
| items | array[GoodwillIndustryItem] | Y | 行业商誉列表 |

GoodwillIndustryItem：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| industry_name | string | Y | 行业名称（如「半导体」） |
| company_count | int | Y | 公司数 |
| goodwill_scale | string | Y | 商誉规模 |
| net_assets | string | Y | 净资产规模 |
| goodwill_to_net_assets_ratio | string | Y | 商誉占净资产比例 |
| net_profit_scale | string | Y | 净利润规模 |

## 调用方法（MCP）

> MCP 工具名 `ft_goodwill_industry`。MCP Streamable HTTP 要求**先 initialize 拿 `Mcp-Session-Id`，再 `tools/call`**，后续请求带该 header。返回 **markdown 表格文本**。

**curl**：

```bash
SID=$(curl -sS -m 10 -D - -o /dev/null -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"t","version":"1"}}}' \
  | grep -i 'mcp-session-id:' | awk '{print $2}' | tr -d '')

curl -sS -m 10 -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -H "Mcp-Session-Id: $SID" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ft_goodwill_industry","arguments":{"date": "20260624"}}}'
```

**Python（`mcp` SDK，自动握手管理 session）**：

```python
import asyncio
from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client

async def main():
    async with streamablehttp_client('<MCP_BASE_URL>') as (r, w, _):
        async with ClientSession(r, w) as s:
            await s.initialize()
            res = await s.call_tool('ft_goodwill_industry', {"date": "20260624"})
            print(res.content[0].text)   # markdown 表格文本

asyncio.run(main())
```
