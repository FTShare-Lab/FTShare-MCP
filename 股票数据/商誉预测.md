# 商誉预测（MCP 工具 `ft_goodwill_predict`）

> **MCP 工具**：`ft_goodwill_predict`（category: `股票数据/财务数据`）。返回 **markdown 表格文本**（非结构化 JSON）。输入参数 / 输出参数 / 数据样例对照 [ftshare-doc](../../ftshare-doc/api-doc) 原接口。

- 描述：按报告期查询 A 股个股商誉减值预期/业绩预告明细，返回业绩变动说明、预告期间、最新商誉、上期商誉、预告净利润下限/上限、业绩变动幅度下限/上限、上年同期净利润、公告日期、所属市场。数据来源于商誉预测表。响应为 `PaginatedApiResponse` 分页信封，data 含 `pageNum/pageSize/total/pages/records`。
- 数据范围：按报告期查询（`date` 必填，如 `20250331`）
- 单次限量：内存分页，默认 `page=1`、`page_size=50`、最大 200
- 提示：
  - `date` 必填，报告期通常为季末/年末。
  - 对应「商誉减值预期/业绩预告」口径，非实际减值（实际减值见 `goodwill/stock-impairment`）。
  - 多数字段为 `Option<String>`，缺失返回 null。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| date | string | Y | 报告期，如 20250331 |
| page | int | N | 页码，默认 1 |
| page_size | int | N | 每页条数，默认 50、最大 200 |

## 输出参数

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| code | int | Y | 状态码：0 成功 |
| message | string | Y | 状态信息 |
| data | object | Y | PaginatedData（见下） |

data ：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| pageNum | int | Y | 当前页码 |
| pageSize | int | Y | 每页条数 |
| total | int | Y | 总记录数 |
| pages | int | Y | 总页数 |
| records | array[GoodwillPredictItem] | Y | 个股商誉预测列表 |

GoodwillPredictItem：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| seq | int | Y | 序号 |
| security_code | string | Y | 证券代码 |
| security_name | string | Y | 证券简称 |
| perform_change_explain | string | Y | 业绩变动说明（如「业绩预增」） |
| predict_period | string | Y | 预告期间（如 20251231） |
| newest_goodwill | string | Y | 最新商誉 |
| goodwill_previous | string | Y | 上期商誉 |
| predict_netprofit_lower | string | Y | 预告净利润下限 |
| predict_netprofit_upper | string | Y | 预告净利润上限 |

| notice_date | string | Y | 公告日期 |
| trade_market | string | Y | 所属市场（如「主板」） |

## 调用方法（MCP）

> MCP 工具名 `ft_goodwill_predict`。MCP Streamable HTTP 要求**先 initialize 拿 `Mcp-Session-Id`，再 `tools/call`**，后续请求带该 header。返回 **markdown 表格文本**。

**curl**：

```bash
SID=$(curl -sS -m 10 -D - -o /dev/null -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"t","version":"1"}}}' \
  | grep -i 'mcp-session-id:' | awk '{print $2}' | tr -d '')

curl -sS -m 10 -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -H "Mcp-Session-Id: $SID" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ft_goodwill_predict","arguments":{"date": "20260624"}}}'
```

**Python（`mcp` SDK，自动握手管理 session）**：

```python
import asyncio
from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client

async def main():
    async with streamablehttp_client('<MCP_BASE_URL>') as (r, w, _):
        async with ClientSession(r, w) as s:
            await s.initialize()
            res = await s.call_tool('ft_goodwill_predict', {"date": "20260624"})
            print(res.content[0].text)   # markdown 表格文本

asyncio.run(main())
```
