# 机构参与度（MCP 工具 `ft_stock_comment_org_participate_em`）

> **MCP 工具**：`ft_stock_comment_org_participate_em`（category: `股票数据/特色数据`）。返回 **markdown 表格文本**（非结构化 JSON）。输入参数 / 输出参数 / 数据样例对照 [ftshare-doc](../../ftshare-doc/api-doc) 原接口。

- 描述：查询单只 A 股的「机构参与度」历史序列。机构参与度来自千股千评主表，反映机构资金对该标的的参与活跃程度。按交易日返回该标的历史机构参与度。响应为 `ApiResponse` 信封（code/message/data），data 含 symbol、total 及 items 数组。
- 数据范围：最新一期快照（按标的返回历史序列；以服务端返回为准）
- 单次限量：单标的历史全量返回，无分页参数
- 提示：
  - `symbol` 必填，格式为纯数字代码（如 `600000`），无交易所后缀。
  - 返回的是该标的历史机构参与度序列，每行一个交易日。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| symbol | string | Y | 证券代码（纯数字，如 600000） |

## 输出参数

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| code | int | Y | 状态码：0 成功 |
| message | string | Y | 状态信息 |
| data | object | Y | 机构参与度数据（见下） |

data ：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| symbol | string | Y | 证券代码（回显） |
| total | int | Y | 序列总条数 |
| items | array[StockCommentOrgParticipateItem] | Y | 机构参与度序列 |

StockCommentOrgParticipateItem：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| trade_date | string | Y | 交易日（如 2026-05-20） |
| org_participate | string | Y | 机构参与度 |

## 调用方法（MCP）

> MCP 工具名 `ft_stock_comment_org_participate_em`。MCP Streamable HTTP 要求**先 initialize 拿 `Mcp-Session-Id`，再 `tools/call`**，后续请求带该 header。返回 **markdown 表格文本**。

**curl**：

```bash
SID=$(curl -sS -m 10 -D - -o /dev/null -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"t","version":"1"}}}' \
  | grep -i 'mcp-session-id:' | awk '{print $2}' | tr -d '')

curl -sS -m 10 -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -H "Mcp-Session-Id: $SID" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ft_stock_comment_org_participate_em","arguments":{"symbol": "600584"}}}'
```

**Python（`mcp` SDK，自动握手管理 session）**：

```python
import asyncio
from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client

async def main():
    async with streamablehttp_client('<MCP_BASE_URL>') as (r, w, _):
        async with ClientSession(r, w) as s:
            await s.initialize()
            res = await s.call_tool('ft_stock_comment_org_participate_em', {"symbol": "600584"})
            print(res.content[0].text)   # markdown 表格文本

asyncio.run(main())
```
