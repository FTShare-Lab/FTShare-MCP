# 机构持股明细（MCP 工具 `ft_get_stock_institution_holdings_detail`）

> **MCP 工具**：`ft_get_stock_institution_holdings_detail`（category: `股票数据/基础数据`）。返回 **markdown 表格文本**（非结构化 JSON）。输入参数 / 输出参数 / 数据样例对照 [ftshare-doc](../../ftshare-doc/api-doc) 原接口。

- 描述：查询 A 股单票机构持仓详细信息。按股票代码 + 年份 + 报告期 + 机构类型查询，返回该标的在指定报告期被各机构持仓的明细：机构名称、持股数量、市值、净值比例、流通比例、总股本比例等。`stock_code`、`report_type`、`inst_type`、`year` 均必填。响应为 `PaginatedResponse` 分页结构（items/total_pages/total_items）。
- 数据范围：按报告期，以服务端返回为准
- 单次限量：默认 `page=1`、`page_size=20`
- 提示：
  - `report_type` 取值：q1 / q2 / q3 / annual / announcement。
  - `inst_type` 取值：all_inst（返回全部）/ fund / qfii / insurance / social_security / securities / trust / other，非 all_inst 时按机构类型过滤。
  - 缺少必填参数时返回错误。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| stock_code | string | Y | 股票代码 |
| year | int | Y | 年份 |
| report_type | ReportType | Y | 报告类型：q1 / q2 / q3 / annual / announcement |
| inst_type | InstitutionType | Y | 机构类型：all_inst / fund / qfii / insurance / social_security / securities / trust / other |
| page | int | N | 页码，默认 1 |
| page_size | int | N | 每页条数，默认 20 |

## 输出参数

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| items | array[StockInstitutionHoldingsDetail] | Y | 机构持仓明细列表 |
| total_pages | int | Y | 总页数 |
| total_items | int | Y | 总记录数 |

StockInstitutionHoldingsDetail：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| stock_code | string | Y | 股票代码 |
| stock_name | string | Y | 股票名称 |
| end_date | int | Y | 报告期截止日 YYYYMMDD |
| report_type | string | Y | 报告类型 |
| year | int | Y | 年份 |
| inst_type | string | Y | 机构类型 |
| inst_id | string | Y | 机构 ID |
| inst_main_id | string | Y | 机构主编码 |
| inst_name | string | Y | 机构名称 |
| hld_shares | decimal | Y | 持股数量 |

| total_share_ratio | decimal | Y | 总股本比例 |

## 调用方法（MCP）

> MCP 工具名 `ft_get_stock_institution_holdings_detail`。MCP Streamable HTTP 要求**先 initialize 拿 `Mcp-Session-Id`，再 `tools/call`**，后续请求带该 header。返回 **markdown 表格文本**。

**curl**：

```bash
SID=$(curl -sS -m 10 -D - -o /dev/null -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"t","version":"1"}}}' \
  | grep -i 'mcp-session-id:' | awk '{print $2}' | tr -d '')

curl -sS -m 10 -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -H "Mcp-Session-Id: $SID" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ft_get_stock_institution_holdings_detail","arguments":{"stock_code": "600519.SH", "year": 2024, "report_type": "annual", "inst_type": "fund"}}}'
```

**Python（`mcp` SDK，自动握手管理 session）**：

```python
import asyncio
from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client

async def main():
    async with streamablehttp_client('<MCP_BASE_URL>') as (r, w, _):
        async with ClientSession(r, w) as s:
            await s.initialize()
            res = await s.call_tool('ft_get_stock_institution_holdings_detail', {"stock_code": "600519.SH", "year": 2024, "report_type": "annual", "inst_type": "fund"})
            print(res.content[0].text)   # markdown 表格文本

asyncio.run(main())
```
