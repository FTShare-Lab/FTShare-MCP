# 涨跌停价（MCP 工具 `ft_stk_limit`）

> **MCP 工具**：`ft_stk_limit`（category: `股票数据/行情数据`）。返回 **markdown 表格文本**（非结构化 JSON）。输入参数 / 输出参数 / 数据样例对照 [ftshare-doc](../../ftshare-doc/api-doc) 原接口。

- 描述：每日涨跌停价格接口（Tushare `stk_limit` 风格）。支持多种查询模式：① 按 `trade_date` 查单日全市场（可用 `instrument_type` 过滤 stock/etf/cb，不传返回三类、排除指数）；② 按 `symbol`（或 `symbol_id`+`market_id`）+ `start_date`/`end_date` 查单票历史区间。返回每个标的每个交易日的昨收、涨停价、跌停价。响应为 `StkLimitItem` 列表（含 `trade_date`/`ts_code`/`instrument_type`/`pre_close`/`up_limit`/`down_limit`）。
- 数据范围：按交易日（单日全市场快照或单票历史区间）
- 单次限量：分页返回，默认 `page=1`、`page_size=50`、最大 500
- 提示：
  - `instrument_type` 取值 `stock` / `etf` / `cb`；不传返回三类（排除指数）。
  - `symbol` 支持 `000001.SZ` / `000001.XSHE` / `600519.SH` 等格式；存在 `symbol` 时忽略 `symbol_id`+`market_id`。
  - 区间查询（`start_date`/`end_date`）需配合 `symbol` 或 `symbol_id`+`market_id`。
  - `page` 默认 1，`page_size` 默认 50、最大 500。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| instrument_type | string | N | 标的类型：stock / etf / cb；不传返回三类（排除指数） |
| symbol | string | N | 标的代码，支持 000001.SZ / 000001.XSHE / 600519.SH 等；存在时忽略 symbol_id+market_id |
| symbol_id | int32 | N | 标的 ID，与 market_id 配合使用 |
| market_id | int16 | N | 市场 ID，与 symbol_id 配合使用 |
| trade_date | int32 | N | 交易日 YYYYMMDD，查单日全市场时使用 |
| start_date | int32 | N | 单票历史区间起始日 YYYYMMDD（需配 symbol 或 symbol_id+market_id） |
| end_date | int32 | N | 单票历史区间结束日 YYYYMMDD |
| page | uint32 | N | 页码，默认 1 |
| page_size | uint32 | N | 每页条数，默认 50，最大 500 |

## 输出参数

StkLimitItem：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| trade_date | int32 | Y | 交易日（YYYYMMDD） |
| ts_code | string | Y | 标的代码（如 000001.SZ） |
| instrument_type | string | Y | 标的类型：stock / etf / cb |
| pre_close | decimal | N | 昨收价 |
| up_limit | decimal | N | 涨停价 |
| down_limit | decimal | N | 跌停价 |

## 调用方法（MCP）

> MCP 工具名 `ft_stk_limit`。MCP Streamable HTTP 要求**先 initialize 拿 `Mcp-Session-Id`，再 `tools/call`**，后续请求带该 header。返回 **markdown 表格文本**。

**curl**：

```bash
SID=$(curl -sS -m 10 -D - -o /dev/null -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"t","version":"1"}}}' \
  | grep -i 'mcp-session-id:' | awk '{print $2}' | tr -d '')

curl -sS -m 10 -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -H "Mcp-Session-Id: $SID" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ft_stk_limit","arguments":{}}}'
```

**Python（`mcp` SDK，自动握手管理 session）**：

```python
import asyncio
from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client

async def main():
    async with streamablehttp_client('<MCP_BASE_URL>') as (r, w, _):
        async with ClientSession(r, w) as s:
            await s.initialize()
            res = await s.call_tool('ft_stk_limit', {})
            print(res.content[0].text)   # markdown 表格文本

asyncio.run(main())
```
