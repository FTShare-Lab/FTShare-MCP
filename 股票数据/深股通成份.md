# 深股通成份（MCP 工具 `ft_sz_hk_stock_connect_members`）

> **MCP 工具**：`ft_sz_hk_stock_connect_members`（category: `股票数据/基础数据`）。返回 **markdown 表格文本**（非结构化 JSON）。输入参数 / 输出参数 / 数据样例对照 [ftshare-doc](../../ftshare-doc/api-doc) 原接口。

- 描述：获取深股通成份列表（即通过港股市场可买的深交所 A 股标的）。无入参、无分页，返回内存快照 `Arc<[StockConnectMember]>`，每项仅含成份标的代码（`SymbolKey`）。该数据由后台定时刷新，调用时返回最新成份名单。
- 数据范围：无时间维度，最新成份名单
- 单次限量：无入参、无分页，一次返回全部成份标的
- 提示：
  - 对外 v1 仅暴露 GET（底层 v0 同时支持 GET+POST）。
  - 成份名单以最新快照为准，无历史变更序列。
  - 与「沪股通成份」（`sh-hk-stock-connect-members`）互斥，分别覆盖深市/沪市 A 股通道。

## 输入参数

无（无需传参）。

## 输出参数

StockConnectMember：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| symbol | object | Y | 成份标的代码（SymbolKey：`symbol` + `market_id`） |

## 调用方法（MCP）

> MCP 工具名 `ft_sz_hk_stock_connect_members`。MCP Streamable HTTP 要求**先 initialize 拿 `Mcp-Session-Id`，再 `tools/call`**，后续请求带该 header。返回 **markdown 表格文本**。

**curl**：

```bash
SID=$(curl -sS -m 10 -D - -o /dev/null -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"t","version":"1"}}}' \
  | grep -i 'mcp-session-id:' | awk '{print $2}' | tr -d '')

curl -sS -m 10 -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -H "Mcp-Session-Id: $SID" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ft_sz_hk_stock_connect_members","arguments":{}}}'
```

**Python（`mcp` SDK，自动握手管理 session）**：

```python
import asyncio
from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client

async def main():
    async with streamablehttp_client('<MCP_BASE_URL>') as (r, w, _):
        async with ClientSession(r, w) as s:
            await s.initialize()
            res = await s.call_tool('ft_sz_hk_stock_connect_members', {})
            print(res.content[0].text)   # markdown 表格文本

asyncio.run(main())
```
