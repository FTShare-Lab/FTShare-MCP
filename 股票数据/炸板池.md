# 炸板池（MCP 工具 `ft_limit_up_break_pool`）

> **MCP 工具**：`ft_limit_up_break_pool`（category: `股票数据/打板专题数据`）。返回 **markdown 表格文本**（非结构化 JSON）。输入参数 / 输出参数 / 数据样例对照 [ftshare-doc](../../ftshare-doc/api-doc) 原接口。

- 描述：查询当日炸板股池。从内存涨跌停事件状态（`LIMIT_EVENT_MAP`）筛选 `limit_up_break` 非空且当前状态非涨停（`status != limit_up`）的股票，即曾经封涨停但已被打开的标的，返回涨停相关字段。已过滤非股票标的（可转债等）。返回裸数组（非信封）。
- 数据范围：当日实时快照，无历史序列
- 单次限量：无分页，全量返回当日炸板股池
- 提示：
  - 仅返回当日封板后被打开（`limit_up_break` 非空）且当前已不在涨停状态的股票。
  - 当前仍在涨停的股票不在炸板池中（请用「涨停池」）。
  - 返回为裸 JSON 数组，无 code/message 信封。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| - | - | - | 无入参 |

## 输出参数

LimitUpBreakPoolItem：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| symbol | string | Y | 标的代码 |
| trade_date | string | Y | 交易日 YYYYMMDD |
| ready | bool | Y | 当日涨跌停价格是否已就绪 |
| status | string | Y | 当前状态：normal / limit_up / limit_down |
| limit_up_price | decimal | Y | 涨停价，单位元 |
| limit_up_enter | array[string] | Y | 进入涨停的时间点数组，格式 HH:mm:ss |
| limit_up_break | array[string] | Y | 炸板时间数组，格式 HH:mm:ss |
| first_limit_up_time | string | Y | 首次封涨停时间 HH:mm:ss，无封板记录时为 null |
| limit_up_break_count | int | Y | 炸板次数 |

## 调用方法（MCP）

> MCP 工具名 `ft_limit_up_break_pool`。MCP Streamable HTTP 要求**先 initialize 拿 `Mcp-Session-Id`，再 `tools/call`**，后续请求带该 header。返回 **markdown 表格文本**。

**curl**：

```bash
SID=$(curl -sS -m 10 -D - -o /dev/null -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"t","version":"1"}}}' \
  | grep -i 'mcp-session-id:' | awk '{print $2}' | tr -d '')

curl -sS -m 10 -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -H "Mcp-Session-Id: $SID" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ft_limit_up_break_pool","arguments":{}}}'
```

**Python（`mcp` SDK，自动握手管理 session）**：

```python
import asyncio
from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client

async def main():
    async with streamablehttp_client('<MCP_BASE_URL>') as (r, w, _):
        async with ClientSession(r, w) as s:
            await s.initialize()
            res = await s.call_tool('ft_limit_up_break_pool', {})
            print(res.content[0].text)   # markdown 表格文本

asyncio.run(main())
```
