# 现金流支持股票代码（MCP 工具 `ft_get_cashflow_stock_code`）

> **MCP 工具**：`ft_get_cashflow_stock_code`（category: `股票数据/财务数据`）。返回 **markdown 表格文本**（非结构化 JSON）。输入参数 / 输出参数 / 数据样例对照 [ftshare-doc](../../ftshare-doc/api-doc) 原接口。

- 描述：获取所有有现金流量表数据（`ashare_cashflow`）的股票代码与名称列表，**不分页**，一次返回全量。用于前端在调 `finance/cashflow` 前枚举可查询的标的。请求体为空。
- 数据范围：最新快照（当前 `ashare_cashflow` 表中所有出现过的股票代码集合）
- 单次限量：无分页，一次性返回全部支持标的
- 提示：
  - 无入参，请求体可留空。
  - 返回 `stock_code` 格式为 6 位数字 + 交易所后缀（如 `000001.SZ`）。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| - | - | - | 空请求体（`CashflowStockCodeRequest {}`） |

## 输出参数

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| items | array | Y | 股票代码与名称列表 |

CashflowStockCodeItem：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| stock_code | string | Y | A 股代码（如 000001.SZ） |
| stock_name | string | Y | 股票名称 |

## 调用方法（MCP）

> MCP 工具名 `ft_get_cashflow_stock_code`。MCP Streamable HTTP 要求**先 initialize 拿 `Mcp-Session-Id`，再 `tools/call`**，后续请求带该 header。返回 **markdown 表格文本**。

**curl**：

```bash
SID=$(curl -sS -m 10 -D - -o /dev/null -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"t","version":"1"}}}' \
  | grep -i 'mcp-session-id:' | awk '{print $2}' | tr -d '')

curl -sS -m 10 -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -H "Mcp-Session-Id: $SID" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ft_get_cashflow_stock_code","arguments":{}}}'
```

**Python（`mcp` SDK，自动握手管理 session）**：

```python
import asyncio
from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client

async def main():
    async with streamablehttp_client('<MCP_BASE_URL>') as (r, w, _):
        async with ClientSession(r, w) as s:
            await s.initialize()
            res = await s.call_tool('ft_get_cashflow_stock_code', {})
            print(res.content[0].text)   # markdown 表格文本

asyncio.run(main())
```
