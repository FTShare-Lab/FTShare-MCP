# 盘前数据（MCP 工具 `ft_stk_premarket`）

> **MCP 工具**：`ft_stk_premarket`（category: `股票数据/特色数据`）。返回 **markdown 表格文本**（非结构化 JSON）。输入参数 / 输出参数 / 数据样例对照 [ftshare-doc](../../ftshare-doc/api-doc) 原接口。

- 描述：每日股本（盘前）数据接口（Tushare `stk_premarket` 风格）。支持多种查询模式：① 按 `trade_date` 查单日全市场；② 按 `ts_code` + `start_date`/`end_date` 查单票历史区间。返回每个标的每个交易日的总股本、A 股流通股本、昨收、涨跌停价、价格（当日为预测开盘价、历史为收盘价）及据此计算的总市值与流通市值。响应为 `StkPremarketItem` 列表。
- 数据范围：按交易日（单日全市场快照或单票历史区间）
- 单次限量：分页返回，默认 `page=1`、`page_size=50`、最大 500
- 提示：
  - `ts_code` 支持 `000001.SZ` / `600519.SH` 等格式。
  - 区间查询（`start_date`/`end_date`）需配合 `ts_code`。
  - `price` 字段：当日为预测开盘价，历史为收盘价。
  - `total_mv` = `totshare_num` × `price`；`float_mv` = `ashare_circ_num` × `price`。
  - `page` 默认 1，`page_size` 默认 50、最大 500。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| ts_code | string | N | 股票代码，如 000001.SZ / 600519.SH |
| trade_date | int32 | N | 交易日 YYYYMMDD，查单日全市场时使用 |
| start_date | int32 | N | 单票历史区间起始日 YYYYMMDD（需配 ts_code） |
| end_date | int32 | N | 单票历史区间结束日 YYYYMMDD |
| page | uint32 | N | 页码，默认 1 |
| page_size | uint32 | N | 每页条数，默认 50，最大 500 |

## 输出参数

StkPremarketItem：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| trade_date | int32 | Y | 交易日（YYYYMMDD） |
| ts_code | string | Y | 标的代码（如 000001.SZ） |
| totshare_num | decimal | Y | 总股本（股） |
| ashare_circ_num | decimal | Y | A 股流通股本（股） |
| pre_close | decimal | N | 昨收价 |
| up_limit | decimal | N | 涨停价 |
| down_limit | decimal | N | 跌停价 |
| price | decimal | N | 当日为预测开盘价，历史为收盘价 |
| total_mv | decimal | N | 总市值（元）= totshare_num × price |
| float_mv | decimal | N | 流通市值（元）= ashare_circ_num × price |

## 调用方法（MCP）

> MCP 工具名 `ft_stk_premarket`。MCP Streamable HTTP 要求**先 initialize 拿 `Mcp-Session-Id`，再 `tools/call`**，后续请求带该 header。返回 **markdown 表格文本**。

**curl**：

```bash
SID=$(curl -sS -m 10 -D - -o /dev/null -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"t","version":"1"}}}' \
  | grep -i 'mcp-session-id:' | awk '{print $2}' | tr -d '')

curl -sS -m 10 -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -H "Mcp-Session-Id: $SID" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ft_stk_premarket","arguments":{}}}'
```

**Python（`mcp` SDK，自动握手管理 session）**：

```python
import asyncio
from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client

async def main():
    async with streamablehttp_client('<MCP_BASE_URL>') as (r, w, _):
        async with ClientSession(r, w) as s:
            await s.initialize()
            res = await s.call_tool('ft_stk_premarket', {})
            print(res.content[0].text)   # markdown 表格文本

asyncio.run(main())
```
