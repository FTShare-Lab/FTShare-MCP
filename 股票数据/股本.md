# 股本（MCP 工具 `ft_get_stock_share_handler`）

> **MCP 工具**：`ft_get_stock_share_handler`（category: `股票数据/基础数据`）。返回 **markdown 表格文本**（非结构化 JSON）。输入参数 / 输出参数 / 数据样例对照 [ftshare-doc](../../ftshare-doc/api-doc) 原接口。

- 描述：获取单票指定日期股本信息。返回该标的在指定日期的总股本、A 股流通/限售/无限售股本、B 股股本、H 股股本、境外上市股本、流通股本合计等字段。`stock_code` 与 `date` 均为必填，`date` 格式 YYYYMMDD。响应为 `PaginatedResponse` 分页结构（items/total_pages/total_items），命中时仅返回 1 条。
- 数据范围：按指定日期快照，以服务端返回为准
- 单次限量：单标的单日最多 1 条
- 提示：
  - `stock_code` 与 `date` 均必填，`date` 格式 YYYYMMDD。
  - 未命中数据时 items 为空数组。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| stock_code | string | Y | 股票代码 |
| date | string | Y | 日期 YYYYMMDD |

## 输出参数

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| items | array[GetStockShareResponse] | Y | 股本信息列表 |
| total_pages | int | Y | 总页数 |
| total_items | int | Y | 总记录数 |

GetStockShareResponse：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| stock_code | string | Y | 股票代码 |
| stock_name | string | Y | 股票名称 |
| totshare_num | decimal | Y | 总股本 |
| ashare_circ_num | decimal | Y | A 股流通股本 |
| ashare_circ_limit_num | decimal | Y | A 股限售流通股本 |
| ashare_circ_unlimit_num | decimal | Y | A 股无限售流通股本 |
| bshare_num | decimal | Y | B 股股本 |
| bshare_circ_num | decimal | Y | B 股流通股本 |
| bshare_uncirc_num | decimal | Y | B 股非流通股本 |
| hshare_num | decimal | Y | H 股股本 |
| osshare_num | decimal | Y | 境外上市股本 |
| share_circ_num | decimal | Y | 流通股本合计 |

## 调用方法（MCP）

> MCP 工具名 `ft_get_stock_share_handler`。MCP Streamable HTTP 要求**先 initialize 拿 `Mcp-Session-Id`，再 `tools/call`**，后续请求带该 header。返回 **markdown 表格文本**。

**curl**：

```bash
SID=$(curl -sS -m 10 -D - -o /dev/null -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"t","version":"1"}}}' \
  | grep -i 'mcp-session-id:' | awk '{print $2}' | tr -d '')

curl -sS -m 10 -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -H "Mcp-Session-Id: $SID" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ft_get_stock_share_handler","arguments":{"stock_code": "600519.SH", "date": "20260624"}}}'
```

**Python（`mcp` SDK，自动握手管理 session）**：

```python
import asyncio
from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client

async def main():
    async with streamablehttp_client('<MCP_BASE_URL>') as (r, w, _):
        async with ClientSession(r, w) as s:
            await s.initialize()
            res = await s.call_tool('ft_get_stock_share_handler', {"stock_code": "600519.SH", "date": "20260624"})
            print(res.content[0].text)   # markdown 表格文本

asyncio.run(main())
```
