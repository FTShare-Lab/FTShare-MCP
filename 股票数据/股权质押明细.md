# 股权质押明细（MCP 工具 `ft_stock_pledge_detail`）

> **MCP 工具**：`ft_stock_pledge_detail`（category: `股票数据/参考数据`）。返回 **markdown 表格文本**（非结构化 JSON）。输入参数 / 输出参数 / 数据样例对照 [ftshare-doc](../../ftshare-doc/api-doc) 原接口。

- 描述：股权质押个股详细，返回每只股票的报告日期、质押比例、质押笔数、质押市值、质押无限售股数、质押限售股数、较上年变动等。数据来源于中国证券登记结算公司（CSDC）`ashare_pledgecsdc` 表。支持两种模式：① 指定 `stock_code` 查询单只股票的质押历史；② 不传 `stock_code` 且 `is_last=true` 返回所有股票最新一期的质押数据。响应为 `PaginatedResponse` 信封（items/total_pages/total_items），带瞬态失败缓存。
- 数据范围：以服务端返回为准
- 单次限量：SQL 分页，默认 `page=1`、`page_size=50`
- 提示：
  - 不传 `stock_code` 时必须 `is_last=true`，否则报错「请提供 stock_code 或设置 is_last=true」。
  - `trade_code` 统一为 `symbol.suffix` 格式（如 `600001.SH`）。
  - 金额/股数字段序列化为字符串（保留精度）。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| stock_code | string | N | 股票代码（symbol.suffix 格式或纯代码）；不传时需配合 `is_last=true` |
| is_last | bool | N | 是否仅获取最新一期（无 `stock_code` 时必填为 true） |
| page | int | N | 页码，默认 1 |
| page_size | int | N | 每页条数，默认 50 |

## 输出参数

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| items | array[PledgeStockHolderResponse] | Y | 个股质押明细列表 |
| total_pages | int | Y | 总页数 |
| total_items | int | Y | 总记录数（未分页前） |

PledgeStockHolderResponse：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| trade_code | string | Y | 股票代码，如 `600001.SH` |
| stock_name | string | Y | 股票名称 |
| report_date | string | Y | 报告日期 |
| pledge_ratio | string | Y | 质押比例（%） |
| pledge_n | int | Y | 质押笔数 |
| pledge_market_value | string | Y | 质押市值（元） |
| pledge_unlimitn | string | Y | 质押无限售股数 |
| pledge_limitn | string | Y | 质押限售股数 |
| last_year_fluctuation | string | Y | 较上年变动（%） |

## 调用方法（MCP）

> MCP 工具名 `ft_stock_pledge_detail`。MCP Streamable HTTP 要求**先 initialize 拿 `Mcp-Session-Id`，再 `tools/call`**，后续请求带该 header。返回 **markdown 表格文本**。

**curl**：

```bash
SID=$(curl -sS -m 10 -D - -o /dev/null -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"t","version":"1"}}}' \
  | grep -i 'mcp-session-id:' | awk '{print $2}' | tr -d '')

curl -sS -m 10 -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -H "Mcp-Session-Id: $SID" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ft_stock_pledge_detail","arguments":{"is_last": true, "page": 1, "page_size": 3}}}'
```

**Python（`mcp` SDK，自动握手管理 session）**：

```python
import asyncio
from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client

async def main():
    async with streamablehttp_client('<MCP_BASE_URL>') as (r, w, _):
        async with ClientSession(r, w) as s:
            await s.initialize()
            res = await s.call_tool('ft_stock_pledge_detail', {"is_last": True, "page": 1, "page_size": 3})
            print(res.content[0].text)   # markdown 表格文本

asyncio.run(main())
```
