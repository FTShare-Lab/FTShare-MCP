# 股权质押汇总（MCP 工具 `ft_stock_pledge_summary`）

> **MCP 工具**：`ft_stock_pledge_summary`（category: `股票数据/参考数据`）。返回 **markdown 表格文本**（非结构化 JSON）。输入参数 / 输出参数 / 数据样例对照 [ftshare-doc](../../ftshare-doc/api-doc) 原接口。

- 描述：股权质押 A 股市场总览，返回交易日期、A 股质押总比例、质押公司数量、质押笔数、质押总股数、质押总市值、沪深 300 指数、沪深 300 周涨跌幅等总量指标。数据来源于中国证券登记结算公司（CSDC）`ashare_pledgecsdc` 表与市场行情聚合。请求仅分页参数。响应为 `PaginatedApiResponse` 分页信封，data 含 `pageNum/pageSize/total/pages/records`。
- 数据范围：最新快照（全市场总量）
- 单次限量：内存分页，默认 `page=1`、`page_size=50`、最大 200
- 提示：
  - 请求仅 `page` / `page_size`，无业务筛选。
  - 全市场总量维度，非个股。
  - 金额类字段序列化为字符串（保留精度）。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| page | int | N | 页码，默认 1 |
| page_size | int | N | 每页条数，默认 50、最大 200 |

## 输出参数

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| code | int | Y | 状态码：0 成功 |
| message | string | Y | 状态信息 |
| data | object | Y | PaginatedData（见下） |

data ：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| pageNum | int | Y | 当前页码 |
| pageSize | int | Y | 每页条数 |
| total | int | Y | 总记录数 |
| pages | int | Y | 总页数 |
| records | array[PledgeMarketSummaryResponse] | Y | 市场质押总览列表 |

PledgeMarketSummaryResponse：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| trade_date | string | Y | 交易日期 |
| pledge_total_ratio | string | Y | A 股质押总比例（%） |
| pledge_company_count | int | Y | 质押公司数量 |
| pledge_deal_count | int | Y | 质押笔数 |
| pledge_total_shares | string | Y | 质押总股数（股） |
| pledge_total_market_value | string | Y | 质押总市值（元） |
| hs300_index | string | Y | 沪深 300 指数 |
| hs300_week_change_ratio | string | Y | 沪深 300 周涨跌幅（%） |

## 调用方法（MCP）

> MCP 工具名 `ft_stock_pledge_summary`。MCP Streamable HTTP 要求**先 initialize 拿 `Mcp-Session-Id`，再 `tools/call`**，后续请求带该 header。返回 **markdown 表格文本**。

**curl**：

```bash
SID=$(curl -sS -m 10 -D - -o /dev/null -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"t","version":"1"}}}' \
  | grep -i 'mcp-session-id:' | awk '{print $2}' | tr -d '')

curl -sS -m 10 -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -H "Mcp-Session-Id: $SID" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ft_stock_pledge_summary","arguments":{}}}'
```

**Python（`mcp` SDK，自动握手管理 session）**：

```python
import asyncio
from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client

async def main():
    async with streamablehttp_client('<MCP_BASE_URL>') as (r, w, _):
        async with ClientSession(r, w) as s:
            await s.initialize()
            res = await s.call_tool('ft_stock_pledge_summary', {})
            print(res.content[0].text)   # markdown 表格文本

asyncio.run(main())
```
