# 股票列表（MCP 工具 `ft_get_stock_list`）

> **MCP 工具**：`ft_get_stock_list`（category: `股票数据/基础数据`）。返回 **markdown 表格文本**（非结构化 JSON）。输入参数 / 输出参数 / 数据样例对照 [ftshare-doc](../../ftshare-doc/api-doc) 原接口。

- 描述：获取所有股票代码与名称列表。无入参，返回全量股票的代码（如 `000001.SZ`）与名称。响应为 `StockListResponse`，`items` 为 `StockListItem` 列表。
- 数据范围：全量股票最新代码与名称（快照，无时间维度）
- 单次限量：全量返回，无分页
- 提示：
  - 无入参，直接返回全量股票列表。
  - 不分页，数据量较大时注意单次返回体积。
  - `stock_code` 输出为带交易所后缀格式（如 `000001.SZ`）。

## 输入参数

StockListRequest 无字段。

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| （无） | - | - | 无入参 |

## 输出参数

StockListResponse：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| items | array[StockListItem] | Y | 股票代码与名称列表 |

StockListItem：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| stock_code | string | Y | 股票代码（如 `000001.SZ`） |
| stock_name | string | Y | 股票名称 |

## 调用方法（MCP）

> MCP 工具名 `ft_get_stock_list`。MCP Streamable HTTP 要求**先 initialize 拿 `Mcp-Session-Id`，再 `tools/call`**，后续请求带该 header。返回 **markdown 表格文本**。

**curl**：

```bash
SID=$(curl -sS -m 10 -D - -o /dev/null -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"t","version":"1"}}}' \
  | grep -i 'mcp-session-id:' | awk '{print $2}' | tr -d '')

curl -sS -m 10 -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -H "Mcp-Session-Id: $SID" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ft_get_stock_list","arguments":{}}}'
```

**Python（`mcp` SDK，自动握手管理 session）**：

```python
import asyncio
from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client

async def main():
    async with streamablehttp_client('<MCP_BASE_URL>') as (r, w, _):
        async with ClientSession(r, w) as s:
            await s.initialize()
            res = await s.call_tool('ft_get_stock_list', {})
            print(res.content[0].text)   # markdown 表格文本

asyncio.run(main())
```
