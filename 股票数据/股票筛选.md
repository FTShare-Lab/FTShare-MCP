# 股票筛选（MCP 工具 `ft_stock_filter`）

> **MCP 工具**：`ft_stock_filter`（category: `股票数据/行情数据`）。返回 **markdown 表格文本**（非结构化 JSON）。输入参数 / 输出参数 / 数据样例对照 [ftshare-doc](../../ftshare-doc/api-doc) 原接口。

- 描述：股票实时行情筛选接口。支持按板块/交易所 `board` 与上市日期起点 `listing_date_since` 筛选股票，并返回筛选股票的实时行情（开高低收、涨跌幅、量额、振幅、换手率、市值、PE、累计复权因子、5/10/20/60/120 日及年初至今涨跌幅、涨跌停价等）。响应为 `PaginatedApiResponse<StockFilterItem>` 信封（code/message/data），data 为分页对象（camelCase：pageNum/pageSize/total/pages/records）。
- 数据范围：实时计算（基于实时行情快照，无固定时间维度）
- 单次限量：分页返回，默认 `page=1`，page_size 由后端定
- 提示：
  - `board` 取值：`star`（科创板）/ `chi_next`（创业板）/ `bjse`（北交所）/ `xshg`（沪市）/ `xshe`（深市）/ `main`（主板）。
  - 支持新股（按 `listing_date_since` 筛选）。
  - 大量字段为可选（实时行情未覆盖时为空）。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| board | string | N | 板块/交易所筛选：`star` / `chi_next` / `bjse` / `xshg` / `xshe` / `main` |
| listing_date_since | string | N | 上市日期起点 YYYYMMDD，筛选此后上市的股票 |
| page | uint32 | N | 页码 |
| page_size | uint32 | N | 每页条数 |

## 调用方法（MCP）

> MCP 工具名 `ft_stock_filter`。MCP Streamable HTTP 要求**先 initialize 拿 `Mcp-Session-Id`，再 `tools/call`**，后续请求带该 header。返回 **markdown 表格文本**。

**curl**：

```bash
SID=$(curl -sS -m 10 -D - -o /dev/null -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"t","version":"1"}}}' \
  | grep -i 'mcp-session-id:' | awk '{print $2}' | tr -d '')

curl -sS -m 10 -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -H "Mcp-Session-Id: $SID" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ft_stock_filter","arguments":{}}}'
```

**Python（`mcp` SDK，自动握手管理 session）**：

```python
import asyncio
from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client

async def main():
    async with streamablehttp_client('<MCP_BASE_URL>') as (r, w, _):
        async with ClientSession(r, w) as s:
            await s.initialize()
            res = await s.call_tool('ft_stock_filter', {})
            print(res.content[0].text)   # markdown 表格文本

asyncio.run(main())
```
