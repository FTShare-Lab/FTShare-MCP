# 股票资金流向（MCP 工具 `ft_stock_capital_flows_paginated`）

> **MCP 工具**：`ft_stock_capital_flows_paginated`（category: `股票数据/资金流向数据`）。返回 **markdown 表格文本**（非结构化 JSON）。输入参数 / 输出参数 / 数据样例对照 [ftshare-doc](../../ftshare-doc/api-doc) 原接口。

- 描述：查询股票资金流向，按交易日返回全市场个股资金净流入分档：超大单 / 大单 / 中单 / 小单净流入，并派生主力净流入（= 超大单 + 大单）。传入 `date` 时按 `ts_nanos` 分区日期从 ClickHouse 查询该交易日 15:30 快照，不传时使用当前内存缓存（`CAPITAL_FLOW_MAP`）。响应为 `PaginatedResponse<CapitalFlowResponse>`，经 ft-data-server 层补充标的名称。
- 数据范围：以服务端返回为准（按交易日更新）
- 单次限量：分页返回，默认 `page=1`、`page_size=50`
- 提示：
  - `date` 格式 YYYYMMDD；不传返回最新内存快照。
  - 净流入字段单位为元；主力净流入 = 超大单净流入 + 大单净流入。
  - 响应为 `PaginatedResponse` 结构（items + total_pages + total_items）。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| date | string | N | 查询日期 YYYYMMDD；传入时查询该日 15:30 快照 |
| page | int | N | 页码，从 1 开始，默认 1 |
| page_size | int | N | 每页数量，默认 50 |

## 输出参数

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| items | array[CapitalFlowResponse] | Y | 资金流向列表 |
| total_pages | int | Y | 总页数 |
| total_items | int | Y | 总记录数（未分页前） |

CapitalFlowResponse：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| symbol | string | Y | 标的代码，对外格式 symbol.suffix |
| symbol_name | string | N | 标的名称 |
| net_inflow_extra_large | string | Y | 超大单净流入，单位元 |
| net_inflow_large | string | Y | 大单净流入，单位元 |
| net_inflow_medium | string | Y | 中单净流入，单位元 |
| net_inflow_small | string | Y | 小单净流入，单位元 |
| net_inflow_main | string | Y | 主力净流入，单位元，公式 = 超大单净流入 + 大单净流入 |

## 调用方法（MCP）

> MCP 工具名 `ft_stock_capital_flows_paginated`。MCP Streamable HTTP 要求**先 initialize 拿 `Mcp-Session-Id`，再 `tools/call`**，后续请求带该 header。返回 **markdown 表格文本**。

**curl**：

```bash
SID=$(curl -sS -m 10 -D - -o /dev/null -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"t","version":"1"}}}' \
  | grep -i 'mcp-session-id:' | awk '{print $2}' | tr -d '')

curl -sS -m 10 -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -H "Mcp-Session-Id: $SID" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ft_stock_capital_flows_paginated","arguments":{}}}'
```

**Python（`mcp` SDK，自动握手管理 session）**：

```python
import asyncio
from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client

async def main():
    async with streamablehttp_client('<MCP_BASE_URL>') as (r, w, _):
        async with ClientSession(r, w) as s:
            await s.initialize()
            res = await s.call_tool('ft_stock_capital_flows_paginated', {})
            print(res.content[0].text)   # markdown 表格文本

asyncio.run(main())
```
