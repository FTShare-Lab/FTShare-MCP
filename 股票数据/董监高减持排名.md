# 董监高减持排名（MCP 工具 `ft_stock_ggmx_sell_ranking_handler`）

> **MCP 工具**：`ft_stock_ggmx_sell_ranking_handler`（category: `股票数据/参考数据`）。返回 **markdown 表格文本**（非结构化 JSON）。输入参数 / 输出参数 / 数据样例对照 [ftshare-doc](../../ftshare-doc/api-doc) 原接口。

- 描述：董监高持股变动减持排行。按时间窗口内每股最新 `change_date` 的减持金额降序排行。`time_range` 支持 `1m`/`3m`/`6m`/`1y`/`2y`，默认 `1m`。返回每股窗口内减持总股数、减持总金额（万元）、最新价、涨跌幅、窗口起止日期等。响应为 `PaginatedApiResponse` 信封（code/message/data），data 为分页对象。字段序列化为 camelCase。
- 数据范围：最新一期 / 滚动时间窗口
- 单次限量：分页返回，默认 `page=1`、`page_size` 由后端定
- 提示：
  - `time_range` 默认 `1m`。
  - 数据来源东方财富董监高持股变动明细（同 stock-ggmx）。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| time_range | string | N | 时间范围：1m / 3m / 6m / 1y / 2y，默认 1m，别名 timeRange |
| page | int | N | 页码 |
| page_size | int | N | 每页条数，别名 pageSize |

## 输出参数

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| code | int | Y | 状态码：0 成功 |
| message | string | Y | 状态信息 |
| data | object | Y | 分页对象（见下） |

data ：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| pageNum | int | Y | 当前页码 |
| pageSize | int | Y | 每页条数 |
| total | int | Y | 总记录数 |
| pages | int | Y | 总页数 |
| records | array[StockGgmxRankingItem] | Y | 减持排行列表 |

StockGgmxRankingItem：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| stock_code | string | Y | 股票代码 |
| stock_name | string | Y | 股票简称 |
| total_shares | string | Y | 窗口内总股数 |
| total_amount_wan | string | Y | 窗口内总金额（万元） |
| latest_price | string | N | 最新价 |
| price_change_rate | string | N | 涨跌幅 |
| time_range | string | Y | 时间范围（回显请求参数） |
| start_date | string | Y | 窗口起始日期 YYYY-MM-DD |
| latest_change_date | string | Y | 窗口内该方向命中的最新 change_date |

## 调用方法（MCP）

> MCP 工具名 `ft_stock_ggmx_sell_ranking_handler`。MCP Streamable HTTP 要求**先 initialize 拿 `Mcp-Session-Id`，再 `tools/call`**，后续请求带该 header。返回 **markdown 表格文本**。

**curl**：

```bash
SID=$(curl -sS -m 10 -D - -o /dev/null -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"t","version":"1"}}}' \
  | grep -i 'mcp-session-id:' | awk '{print $2}' | tr -d '')

curl -sS -m 10 -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -H "Mcp-Session-Id: $SID" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ft_stock_ggmx_sell_ranking_handler","arguments":{}}}'
```

**Python（`mcp` SDK，自动握手管理 session）**：

```python
import asyncio
from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client

async def main():
    async with streamablehttp_client('<MCP_BASE_URL>') as (r, w, _):
        async with ClientSession(r, w) as s:
            await s.initialize()
            res = await s.call_tool('ft_stock_ggmx_sell_ranking_handler', {})
            print(res.content[0].text)   # markdown 表格文本

asyncio.run(main())
```
