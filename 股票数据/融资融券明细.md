# 融资融券明细（MCP 工具 `ft_margin_trading_details`）

> **MCP 工具**：`ft_margin_trading_details`（category: `股票数据/两融及转融通`）。返回 **markdown 表格文本**（非结构化 JSON）。输入参数 / 输出参数 / 数据样例对照 [ftshare-doc](../../ftshare-doc/api-doc) 原接口。

- 描述：查询融资融券明细，按交易日返回全市场两融快照（每标的一行）。支持按 `date` 指定交易日；不传 `date` 时使用当前内存快照。底层为 `data-api-server` 的 `margin_trading_details` handler：传入日期时直接读取文件中心对应交易日的 CSV，不传时回退到内存缓存 `MARGIN_TRADE`。响应经 ft-data-server 层转换为分页结构（`PaginatedResponse<MarginTradingDetailResponse>`），并补充标的名称。注意：v1 对外路径只有 `/margin-trading-details`，底层 v0 另有 `/margin-trading-details/paginated` 与非分页变体，两者字段一致、区别仅分页语义。
- 数据范围：以服务端返回为准（按交易日更新）
- 单次限量：分页返回，默认 `page=1`、`page_size=50`
- 提示：
  - `date` 格式 YYYYMMDD；不传返回最新内存快照。
  - 金额字段单位为元，量字段单位为股。
  - 响应为 `PaginatedResponse` 结构（items + total_pages + total_items）。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| date | string | N | 查询日期 YYYYMMDD；不传则使用当前内存快照 |
| page | int | N | 页码，从 1 开始，默认 1 |
| page_size | int | N | 每页数量，默认 50 |

## 输出参数

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| items | array[MarginTradingDetailResponse] | Y | 融资融券明细列表 |
| total_pages | int | Y | 总页数 |
| total_items | int | Y | 总记录数（未分页前） |

MarginTradingDetailResponse：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| date | string | Y | 交易日 YYYY-MM-DD |
| symbol | string | Y | 标的代码，对外格式 symbol.suffix |
| symbol_name | string | N | 标的名称 |
| margin_trading_balance | int | Y | 融资余额，单位元 |
| margin_trading_buying_amount | int | Y | 融资买入额，单位元 |
| margin_trading_repayment_amount | int | Y | 融资偿还额，单位元 |
| securities_lending_balance_volume | int | Y | 融券余量，单位股 |
| securities_lending_repayment_volume | int | Y | 融券偿还量，单位股 |
| securities_lending_selling_volume | int | Y | 融券卖出量，单位股 |
| total_balance | int | Y | 融资融券余额，单位元 |

## 调用方法（MCP）

> MCP 工具名 `ft_margin_trading_details`。MCP Streamable HTTP 要求**先 initialize 拿 `Mcp-Session-Id`，再 `tools/call`**，后续请求带该 header。返回 **markdown 表格文本**。

**curl**：

```bash
SID=$(curl -sS -m 10 -D - -o /dev/null -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"t","version":"1"}}}' \
  | grep -i 'mcp-session-id:' | awk '{print $2}' | tr -d '')

curl -sS -m 10 -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -H "Mcp-Session-Id: $SID" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ft_margin_trading_details","arguments":{}}}'
```

**Python（`mcp` SDK，自动握手管理 session）**：

```python
import asyncio
from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client

async def main():
    async with streamablehttp_client('<MCP_BASE_URL>') as (r, w, _):
        async with ClientSession(r, w) as s:
            await s.initialize()
            res = await s.call_tool('ft_margin_trading_details', {})
            print(res.content[0].text)   # markdown 表格文本

asyncio.run(main())
```
