# 限售解禁（MCP 工具 `ft_stock_unlock_handler`）

> **MCP 工具**：`ft_stock_unlock_handler`（category: `股票数据/参考数据`）。返回 **markdown 表格文本**（非结构化 JSON）。输入参数 / 输出参数 / 数据样例对照 [ftshare-doc](../../ftshare-doc/api-doc) 原接口。

- 描述：按证券代码查询限售解禁批次及股东明细，批次维度分页。每个批次含解禁日、股东户数、解禁股数/当前流通股数/未流通股数、解禁市值、解禁比例（总量/流通）、新股价、限售股份类型、解禁前 20 日/后 20 日涨跌幅、爬取日期、来源，并嵌套该批次下的股东明细列表。数据来源于东方财富（eastmoney）解禁批次表（`em_stock_unlock_batch` + `em_stock_unlock_holder`）。响应为 `PaginatedApiResponse` 分页信封，data 含 `pageNum/pageSize/total/pages/records`。
- 数据范围：以服务端返回为准
- 单次限量：SQL 分页，默认 `page=1`、`page_size=50`
- 提示：
  - `stock_code` 必填，为证券代码。
  - 分页以「批次」为粒度，每个批次 records 内嵌 `holders` 股东明细。
  - 字段序列化为 camelCase（如 `stockCode`、`unlockDate`、`holderCount`）。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| stock_code | string | Y | 证券代码 |
| page | int | N | 页码，默认 1 |
| page_size | int | N | 每页条数，默认 50 |

## 调用方法（MCP）

> MCP 工具名 `ft_stock_unlock_handler`。MCP Streamable HTTP 要求**先 initialize 拿 `Mcp-Session-Id`，再 `tools/call`**，后续请求带该 header。返回 **markdown 表格文本**。

**curl**：

```bash
SID=$(curl -sS -m 10 -D - -o /dev/null -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"t","version":"1"}}}' \
  | grep -i 'mcp-session-id:' | awk '{print $2}' | tr -d '')

curl -sS -m 10 -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -H "Mcp-Session-Id: $SID" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ft_stock_unlock_handler","arguments":{"stock_code": "600519.SH"}}}'
```

**Python（`mcp` SDK，自动握手管理 session）**：

```python
import asyncio
from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client

async def main():
    async with streamablehttp_client('<MCP_BASE_URL>') as (r, w, _):
        async with ClientSession(r, w) as s:
            await s.initialize()
            res = await s.call_tool('ft_stock_unlock_handler', {"stock_code": "600519.SH"})
            print(res.content[0].text)   # markdown 表格文本

asyncio.run(main())
```
