# 集合竞价结果（MCP 工具 `ft_auction_results`）

> **MCP 工具**：`ft_auction_results`（category: `股票数据/特色数据`）。返回 **markdown 表格文本**（非结构化 JSON）。输入参数 / 输出参数 / 数据样例对照 [ftshare-doc](../../ftshare-doc/api-doc) 原接口。

- 描述：返回当日所有股票的盘前集合竞价结果（9:25 开盘集合竞价快照，来自 ClickHouse）：集合竞价开盘价/收盘价（最新价）、最高价、最低价、成交量、成交额、均价（vwap）。支持按 `ts_code` 单标的查询，或按 `trade_date` / `start_date`+`end_date` 区间查询。响应为 `PaginatedApiResponse` 信封（code/message/data），data 含分页字段（pageNum/pageSize/total/pages/records）。
- 数据范围：当日盘前集合竞价快照，支持 `trade_date` / 日期区间查询
- 单次限量：分页，`page` 默认 1，`page_size` 默认 50、最大 200
- 提示：
  - 支持 `ts_code`（单标的）、`trade_date`（单日）、`start_date`+`end_date`（区间）组合查询。
  - 响应为 code/message/data 信封，data 为分页对象（camelCase：pageNum/pageSize/total/pages/records）。
  - `page_size` 上限 200。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| ts_code | string | N | 股票代码，如 000001.XSHE |
| trade_date | string | N | 交易日，格式 YYYYMMDD |
| start_date | string | N | 起始日期，格式 YYYYMMDD |
| end_date | string | N | 结束日期，格式 YYYYMMDD |
| page | int | N | 页码，从 1 开始，默认 1 |
| page_size | int | N | 每页数量，默认 50，最大 200 |

## 输出参数

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| code | int | Y | 状态码：0 成功 |
| message | string | Y | 状态信息 |
| data | object | Y | 分页对象（见下） |

data ：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| pageNum | int | Y | 当前页码 |
| pageSize | int | Y | 每页条数 |
| total | int | Y | 总记录数 |
| pages | int | Y | 总页数 |
| records | array[AuctionResult] | Y | 集合竞价结果列表 |

AuctionResult：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| symbol | string | Y | 标的代码 |
| close | decimal | Y | 集合竞价收盘价（最新价），单位元 |
| open | decimal | Y | 集合竞价开盘价，单位元 |
| high | decimal | Y | 最高价，单位元 |
| low | decimal | Y | 最低价，单位元 |
| volume | int | Y | 成交量 |
| amount | decimal | Y | 成交额，单位元 |
| vwap | decimal | Y | 均价，公式 = 成交额 / 成交量 |

## 调用方法（MCP）

> MCP 工具名 `ft_auction_results`。MCP Streamable HTTP 要求**先 initialize 拿 `Mcp-Session-Id`，再 `tools/call`**，后续请求带该 header。返回 **markdown 表格文本**。

**curl**：

```bash
SID=$(curl -sS -m 10 -D - -o /dev/null -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"t","version":"1"}}}' \
  | grep -i 'mcp-session-id:' | awk '{print $2}' | tr -d '')

curl -sS -m 10 -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -H "Mcp-Session-Id: $SID" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ft_auction_results","arguments":{}}}'
```

**Python（`mcp` SDK，自动握手管理 session）**：

```python
import asyncio
from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client

async def main():
    async with streamablehttp_client('<MCP_BASE_URL>') as (r, w, _):
        async with ClientSession(r, w) as s:
            await s.initialize()
            res = await s.call_tool('ft_auction_results', {})
            print(res.content[0].text)   # markdown 表格文本

asyncio.run(main())
```
