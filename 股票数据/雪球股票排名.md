# 雪球股票排名（MCP 工具 `ft_xueqiu_rank`）

> **MCP 工具**：`ft_xueqiu_rank`（category: `股票数据/特色数据`）。返回 **markdown 表格文本**（非结构化 JSON）。输入参数 / 输出参数 / 数据样例对照 [ftshare-doc](../../ftshare-doc/api-doc) 原接口。

- 描述：查询雪球股票排行榜最新一期快照。来源标识为雪球（xueqiu），支持按榜单组（关注 / 讨论 / 交易）与周期（本周新增 / 最热门）组合查询；返回榜单展示名、核心指标名（关注度/讨论数/交易数）、总记录数与排行条目（排名、统一代码、简称、核心指标值、最新价、雪球原始代码）。ft-data-server 做了带失败退避的内存缓存。响应为 `ApiResponse` 信封（code/message/data）。
- 数据范围：雪球排名最新一期快照（无时间序列）
- 单次限量：分页返回，默认 `page=1`、`page_size=20`，上限 100
- 提示：
  - 三层命名遵循 CLAUDE.md 来源标识约定：v0 `/stock-rank/xueqiu`、client `xueqiu_stock_rank`、v1 `/xueqiu-rank`、对外类型 `XueqiuRank*`。
  - `trade_date` 不传默认取最新一期；格式 `YYYY-MM-DD`。
  - 响应为 `ApiResponse<XueqiuRankResponseData>` 信封结构。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| rank_group | string | N | 榜单组：`follow`（关注）/ `tweet`（讨论）/ `deal`（交易） |
| period | string | N | 周期：`7d`（本周新增）/ `total`（最热门） |
| trade_date | string | N | 交易日期，格式 `YYYY-MM-DD`，不传默认取最新 |
| page | int | N | 页码，从 1 开始，默认 1 |
| page_size | int | N | 每页条数，默认 20，最大 100 |

## 输出参数

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| code | int | Y | 状态码：0 成功 |
| message | string | Y | 状态信息 |
| data | object | Y | 排行数据对象（见下） |

data ：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| trade_date | string | Y | 数据日期 |
| display_name | string | Y | 榜单展示名，如"雪球关注-本周新增-A股" |
| metric_name | string | Y | 核心指标名（关注度/讨论数/交易数） |
| total | int | Y | 总记录数 |
| items | array[XueqiuRankItem] | Y | 排行数据列表 |

XueqiuRankItem：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| rank_no | int | Y | 排名 |
| normalized_symbol | string | Y | 统一股票代码，如 `000001` |
| stock_name | string | Y | 股票简称 |
| metric_value | string | Y | 核心指标值（关注度/讨论数/交易数） |
| latest_price | string | Y | 最新价 |
| raw_symbol | string | Y | 雪球原始代码，如 `SZ000001` |

## 调用方法（MCP）

> MCP 工具名 `ft_xueqiu_rank`。MCP Streamable HTTP 要求**先 initialize 拿 `Mcp-Session-Id`，再 `tools/call`**，后续请求带该 header。返回 **markdown 表格文本**。

**curl**：

```bash
SID=$(curl -sS -m 10 -D - -o /dev/null -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"t","version":"1"}}}' \
  | grep -i 'mcp-session-id:' | awk '{print $2}' | tr -d '')

curl -sS -m 10 -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -H "Mcp-Session-Id: $SID" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ft_xueqiu_rank","arguments":{}}}'
```

**Python（`mcp` SDK，自动握手管理 session）**：

```python
import asyncio
from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client

async def main():
    async with streamablehttp_client('<MCP_BASE_URL>') as (r, w, _):
        async with ClientSession(r, w) as s:
            await s.initialize()
            res = await s.call_tool('ft_xueqiu_rank', {})
            print(res.content[0].text)   # markdown 表格文本

asyncio.run(main())
```
