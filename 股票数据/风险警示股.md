# 风险警示股（MCP 工具 `ft_risk_warning_stocks`）

> **MCP 工具**：`ft_risk_warning_stocks`（category: `股票数据/参考数据`）。返回 **markdown 表格文本**（非结构化 JSON）。输入参数 / 输出参数 / 数据样例对照 [ftshare-doc](../../ftshare-doc/api-doc) 原接口。

- 描述：查询指定交易日的风险警示股票基础名单（ST/*ST 等）。返回当日风险警示股票的代码、名称、风险类型标签。`date` 必填（YYYYMMDD）。响应为 `ApiResponse` 信封（code/message/data），data 含日期、总数、名单列表。
- 数据范围：按交易日，`date` 指定哪一期风险警示名单
- 单次限量：无分页，全量返回该日风险警示名单
- 提示：
  - `date` 必填，格式 YYYYMMDD。
  - 响应为 code/message/data 信封，data 为对象（非分页）。
  - 如需带实时行情，用「风险警示股行情」接口（`/risk-warning-stocks/quotes`）。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| date | string | Y | 交易日，格式 YYYYMMDD |

## 输出参数

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| code | int | Y | 状态码：0 成功 |
| message | string | Y | 状态信息 |
| data | object | Y | 风险警示股数据（见下） |

data ：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| date | string | Y | 交易日 YYYYMMDD |
| total | int | Y | 风险警示股总数 |
| items | array[RiskWarningStockItem] | Y | 风险警示股名单 |

RiskWarningStockItem：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| stock_code | string | Y | 股票代码 |
| stock_name | string | Y | 股票名称 |
| risk_type | string | Y | 风险类型标签，如 ST / *ST |

## 调用方法（MCP）

> MCP 工具名 `ft_risk_warning_stocks`。MCP Streamable HTTP 要求**先 initialize 拿 `Mcp-Session-Id`，再 `tools/call`**，后续请求带该 header。返回 **markdown 表格文本**。

**curl**：

```bash
SID=$(curl -sS -m 10 -D - -o /dev/null -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"t","version":"1"}}}' \
  | grep -i 'mcp-session-id:' | awk '{print $2}' | tr -d '')

curl -sS -m 10 -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -H "Mcp-Session-Id: $SID" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ft_risk_warning_stocks","arguments":{"date": "20260624"}}}'
```

**Python（`mcp` SDK，自动握手管理 session）**：

```python
import asyncio
from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client

async def main():
    async with streamablehttp_client('<MCP_BASE_URL>') as (r, w, _):
        async with ClientSession(r, w) as s:
            await s.initialize()
            res = await s.call_tool('ft_risk_warning_stocks', {"date": "20260624"})
            print(res.content[0].text)   # markdown 表格文本

asyncio.run(main())
```
