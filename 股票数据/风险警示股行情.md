# 风险警示股行情（MCP 工具 `ft_risk_warning_stock_quotes`）

> **MCP 工具**：`ft_risk_warning_stock_quotes`（category: `股票数据/参考数据`）。返回 **markdown 表格文本**（非结构化 JSON）。输入参数 / 输出参数 / 数据样例对照 [ftshare-doc](../../ftshare-doc/api-doc) 原接口。

- 描述：查询指定交易日的风险警示股票名单及其实时行情。每项在「风险警示股」基础上 `flatten` 嵌入 `StockInfo`（实时行情快照），含最新价、开/高/低/前收、涨跌额/涨跌幅、成交量/成交额、均价、委比、多周期涨跌幅、涨停/跌停价及是否封板、市值、流通股本等。`date` 必填（YYYYMMDD）。响应为 `ApiResponse` 信封（code/message/data），data 含日期、总数、名单+行情列表。
- 数据范围：按交易日，`date` 指定哪一期风险警示名单；行情为实时快照
- 单次限量：无分页，全量返回该日风险警示股+行情
- 提示：
  - `date` 必填，格式 YYYYMMDD。
  - 每项 = 风险警示基础字段（stock_code/stock_name/risk_type）+ StockInfo 实时行情字段（扁平展开，非嵌套）。
  - 响应为 code/message/data 信封，data 为对象（非分页）。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| date | string | Y | 交易日，格式 YYYYMMDD |

## 调用方法（MCP）

> MCP 工具名 `ft_risk_warning_stock_quotes`。MCP Streamable HTTP 要求**先 initialize 拿 `Mcp-Session-Id`，再 `tools/call`**，后续请求带该 header。返回 **markdown 表格文本**。

**curl**：

```bash
SID=$(curl -sS -m 10 -D - -o /dev/null -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"t","version":"1"}}}' \
  | grep -i 'mcp-session-id:' | awk '{print $2}' | tr -d '')

curl -sS -m 10 -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -H "Mcp-Session-Id: $SID" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ft_risk_warning_stock_quotes","arguments":{"date": "20260624"}}}'
```

**Python（`mcp` SDK，自动握手管理 session）**：

```python
import asyncio
from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client

async def main():
    async with streamablehttp_client('<MCP_BASE_URL>') as (r, w, _):
        async with ClientSession(r, w) as s:
            await s.initialize()
            res = await s.call_tool('ft_risk_warning_stock_quotes', {"date": "20260624"})
            print(res.content[0].text)   # markdown 表格文本

asyncio.run(main())
```
