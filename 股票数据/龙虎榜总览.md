# 龙虎榜总览（MCP 工具 `ft_abnormal_trading_overview`）

> **MCP 工具**：`ft_abnormal_trading_overview`（category: `股票数据/打板专题数据`）。返回 **markdown 表格文本**（非结构化 JSON）。输入参数 / 输出参数 / 数据样例对照 [ftshare-doc](../../ftshare-doc/api-doc) 原接口。

- 描述：查询龙虎榜总览（异动成交概览），按交易日返回当日上榜标的的总览信息：仅含标的代码与涨跌幅，按交易所分桶（上交所 xshg / 深交所 xshe / 北交所 bjse）。传入 `date` 时直接读取文件中心当日 CSV，不传时回退内存缓存 `ABNORMAL_TRADING_OVERVIEW`。响应为单个 `AbnormalTradingOverviewResponse` 对象（非分页），经 ft-data-server 层补充标的名称。与「龙虎榜明细」的区别：总览不含营业部买卖明细与成交额，仅给上榜标的的涨跌幅清单，适合快速扫描上榜名单。
- 数据范围：以服务端返回为准（按交易日更新）
- 单次限量：无分页，一次返回当日全部上榜标的（按交易所分三组）
- 提示：
  - `date` 格式 YYYYMMDD；不传返回最新内存快照。
  - 响应按交易所分桶：`xshg`（上交所）/ `xshe`（深交所）/ `bjse`（北交所）。
  - 每项仅含涨跌幅，不含买卖营业部明细。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| date | string | N | 查询日期 YYYYMMDD；不传则使用当前内存快照 |

## 输出参数

AbnormalTradingOverviewResponse：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| xshg | array[Item] | Y | 上交所上榜标的列表 |
| xshe | array[Item] | Y | 深交所上榜标的列表 |
| bjse | array[Item] | Y | 北交所上榜标的列表 |

Item（AbnormalTradingOverviewResponseItem）：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| symbol | string | Y | 标的代码，对外格式 symbol.suffix |
| symbol_name | string | N | 标的名称 |
| change_rate | f64 | Y | 涨跌幅 |

## 调用方法（MCP）

> MCP 工具名 `ft_abnormal_trading_overview`。MCP Streamable HTTP 要求**先 initialize 拿 `Mcp-Session-Id`，再 `tools/call`**，后续请求带该 header。返回 **markdown 表格文本**。

**curl**：

```bash
SID=$(curl -sS -m 10 -D - -o /dev/null -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"t","version":"1"}}}' \
  | grep -i 'mcp-session-id:' | awk '{print $2}' | tr -d '')

curl -sS -m 10 -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -H "Mcp-Session-Id: $SID" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ft_abnormal_trading_overview","arguments":{}}}'
```

**Python（`mcp` SDK，自动握手管理 session）**：

```python
import asyncio
from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client

async def main():
    async with streamablehttp_client('<MCP_BASE_URL>') as (r, w, _):
        async with ClientSession(r, w) as s:
            await s.initialize()
            res = await s.call_tool('ft_abnormal_trading_overview', {})
            print(res.content[0].text)   # markdown 表格文本

asyncio.run(main())
```
