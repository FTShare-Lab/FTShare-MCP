# 龙虎榜明细（MCP 工具 `ft_abnormal_trading_details`）

> **MCP 工具**：`ft_abnormal_trading_details`（category: `股票数据/打板专题数据`）。返回 **markdown 表格文本**（非结构化 JSON）。输入参数 / 输出参数 / 数据样例对照 [ftshare-doc](../../ftshare-doc/api-doc) 原接口。

- 描述：查询龙虎榜明细（异动成交详情），按交易日返回当日上榜标的明细：含标的涨跌幅、收盘价、成交额，以及买入额/卖出额前 5 名营业部明细。传入 `date` 时直接读取文件中心当日 CSV（`STRANGE_TRADE`），不传时回退内存缓存 `ABNORMAL_TRADING_DETAILS`。响应为 `Vec<AbnormalTradingDetailResponse>`（非分页，一次返回全部上榜标的），经 ft-data-server 层补充标的名称。
- 数据范围：以服务端返回为准（按交易日更新）
- 单次限量：无入参分页，一次返回当日全部上榜标的
- 提示：
  - `date` 格式 YYYYMMDD；不传返回最新内存快照。
  - `top_buyers` / `top_sellers` 各为前 5 名营业部列表，含买入额/卖出额/净额。
  - 金额字段单位为元。

## 输入参数

| 名称 | 类型 | 必选 | 描述 |
|------|------|------|------|
| date | string | N | 查询日期 YYYYMMDD；不传则使用当前内存快照 |

## 输出参数

AbnormalTradingDetailResponse：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| symbol | string | Y | 标的代码，对外格式 symbol.suffix |
| symbol_name | string | N | 标的名称 |
| change_rate | f64 | Y | 涨跌幅，公式 = 涨跌额 / 前收盘价 |
| close | string | Y | 收盘价，单位元 |
| turnover | string | Y | 成交额，单位元 |
| top_buyers | array[Trader] | Y | 买入额前 5 名营业部 |
| top_sellers | array[Trader] | Y | 卖出额前 5 名营业部 |

Trader：

| 名称 | 类型 | 默认显示 | 描述 |
|------|------|---------|------|
| name | string | Y | 营业部名称 |
| buy | string | Y | 买入额，单位元 |
| sell | string | Y | 卖出额，单位元 |
| net | string | Y | 净额，单位元，公式 = 买入额 - 卖出额 |

## 调用方法（MCP）

> MCP 工具名 `ft_abnormal_trading_details`。MCP Streamable HTTP 要求**先 initialize 拿 `Mcp-Session-Id`，再 `tools/call`**，后续请求带该 header。返回 **markdown 表格文本**。

**curl**：

```bash
SID=$(curl -sS -m 10 -D - -o /dev/null -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -d '{"jsonrpc":"2.0","id":1,"method":"initialize","params":{"protocolVersion":"2025-11-25","capabilities":{},"clientInfo":{"name":"t","version":"1"}}}' \
  | grep -i 'mcp-session-id:' | awk '{print $2}' | tr -d '')

curl -sS -m 10 -X POST <MCP_BASE_URL> \
  -H "Accept: application/json, text/event-stream" -H "Content-Type: application/json" \
  -H "Mcp-Session-Id: $SID" \
  -d '{"jsonrpc":"2.0","id":2,"method":"tools/call","params":{"name":"ft_abnormal_trading_details","arguments":{}}}'
```

**Python（`mcp` SDK，自动握手管理 session）**：

```python
import asyncio
from mcp import ClientSession
from mcp.client.streamable_http import streamablehttp_client

async def main():
    async with streamablehttp_client('<MCP_BASE_URL>') as (r, w, _):
        async with ClientSession(r, w) as s:
            await s.initialize()
            res = await s.call_tool('ft_abnormal_trading_details', {})
            print(res.content[0].text)   # markdown 表格文本

asyncio.run(main())
```
